# 文字修仙游戏第一章 MVP：《旁支入道》

这是一个纯控制台文字修仙游戏原型，当前只实现第一章《旁支入道》。

玩家扮演青岭沈家旁支子弟，在一年内完成 12 个月、每月 3 次行动，共 36 次行动。最终参加青岭沈家家族大比，目标是进入前十。

## 运行方式

```bash
python main.py
```

## 已实现范围

- 玩家创建
- 随机灵根
- 角色属性
- 每月行动
- 古玉瓶基础催熟
- 灵草、灵石、丹药、家族贡献
- 暴露度、心魔值、魔气值、业力值、正道声望
- NPC 好感基础版
- 情意锁低阶版：只作为好感辅助
- 万魂幡隐藏线：只做低阶炼魂
- 家族大比三关评分
- 结局结算
- 简单 JSON 存档读档

## 当前不做

- 正式宗门系统
- 自建宗门
- 怀孕后代
- 高阶双修
- 高阶万魂幡
- 复杂装备系统
- 阵法、灵兽、炼器
- 图形界面

## 文件说明

- `main.py`：控制台入口、主菜单、行动循环。
- `player.py`：玩家数据、创建、序列化。
- `systems.py`：第一章行动系统、月末事件。
- `tournament.py`：家族大比三关评分和结局。
- `data.py`：常量、灵根、NPC、文本数据。
- `save_load.py`：JSON 存档读档。

## 存档

游戏内输入 `S` 可以保存到 `save.json`，输入 `L` 可以从 `save.json` 读取。
