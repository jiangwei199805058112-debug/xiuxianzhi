# 文字修仙游戏第一章 MVP：《旁支入道》v0.1.11

这是一个纯控制台文字修仙游戏原型，当前只实现第一章《旁支入道》。v0.1.11 是盗术系统稳定性与反馈打磨版本，重点增强盗术状态、失败后果、高阶反噬、大比评价和模拟统计；不新增第二章，不做图形界面。

玩家扮演青岭沈家旁支子弟，在一年内完成 12 个月、每月 3 次行动，共 36 次行动。最终参加青岭沈家家族大比，目标是进入前十。

## 运行方式

```bash
python main.py
```

## v0.1.11 已实现范围

- 玩家创建、随机灵根、境界层数、修炼进度、气血、灵力、攻防身法等基础属性。
- 普通灵草、十年份灵草、三十年份灵草、灵石、丹药、家族贡献。
- 暴露度、心魔值、魔气值、业力值、正道声望。
- NPC 好感基础版，名单固定为沈若兰、沈云庭、沈子岳、沈怀安、沈霜、沈墨阳。
- 古玉瓶：第一次照看药田获得；获得后可少量、中量、大量催熟。
- 坊市交易：支持买入基础商品、出售灵草、打听行情；商品包含灵草、丹药、符箓、坊市情报和黑市线索。
- 黑水坊市 v1.1：从 `configs/heishui_market` 读取商品、店铺、NPC、行情事件、刷新与解锁规则；支持店铺解锁、月度刷新、动态价格、购买、出售、寄卖、拍卖、服务消费、风险判定、天机阁情报、风月楼淡出事件和盲盒判定。
- 黑水坊市内容一期扩展：天机阁情报品质、散修黑市高风险商品、风月楼轻量选项、地摊盲盒结果池、月末追踪事件。
- 灵田系统：玩家初始拥有 1 块简陋灵田，可一键种植、一键照料、一键收获，并可升级到 2~3 块；作物按月成长，主要产出炼丹材料和少量灵草。
- 灵田反馈增强：已种植或已经营的灵田在月末可能出现轻量氛围事件，多数只显示文本，少数有微量材料、长势或暴露变化。
- 炼丹炉系统：支持破旧丹炉、青铜丹炉、玄铁丹炉，影响偷偷炼丹时的成功率、丹毒、成丹数量和轻微暴露。
- 炼丹文本增强：炼丹会根据结果显示完美成丹、正常成丹、丹毒、部分失败、完全失败及不同丹炉火候描述。
- 轻量装备系统：支持武器、法袍、靴子、护符 4 个槽位；装备只提供克制属性加成，不做强化、套装、镶嵌、耐久或炼器。
- 装备对比：更换装备前显示当前装备与选择装备的属性差异，例如攻击、防御、身法、道心、神识、暴露度、心魔值的变化。
- 装备自然获取：普通坊市可购买基础装备，百药山和家族杂务有低概率获得旧装备，黑水盲盒有极低概率夹带来历不明旧物，大比结算后可能发放奖励装备。
- 新手提示：第 1 月和第 2 月各显示一次简短提醒，说明稳妥修炼、冲榜资源和风险后果。
- 黑水风险提示：首次进入黑水坊市显示风险提示，暴露、业力或追踪标记偏高时显示额外警告。
- 盗术系统：修炼准备中新增“盗术/旁门技艺”，可修习盗术、顺手牵羊、夜探摊位、盗取情报，并按盗术等级逐步解锁窃取功法、修为、气运、机缘、因果、寿元和 NPC 传承。
- 盗术分层：盗术不足 10 只能做低阶小偷小摸；10、25、45、65、80、95 分别解锁更高风险内容，高阶盗术不作为稳定收益按钮。
- 盗术失败处理：偷窃败露后可赔偿、拒赔或强行脱身。赔偿损失灵石但后患较轻；拒赔会损声望并结仇；强行脱身会明显增加暴露、业力和追踪风险。
- 盗术月末反噬：失主认出、坊市盗修传闻、黑水盯梢、气运反噬、因果缠身、寿元阴债、传承原主追查等会根据偷窃次数、失败、声望、结仇、业力和暴露触发。
- 盗术大比评价：普通盗术成功可提供少量情报或隐藏准备，偷功法/偷修为可提升战力或修为评分；高暴露、高业力、高失败和结仇会带来“旁门痕迹过重”“手脚不干净”等负面标记。
- 盗术反馈打磨：状态中显示盗术等级、经验、偷窃次数、成功/失败、赔偿/拒赔/强行脱身、结仇、高阶触发、高阶成功和月末反噬次数。
- 盗术文本增强：基础偷窃、夜探摊位、盗情报、偷功法、偷修为、偷气运、截胡机缘、偷因果、偷寿元和偷 NPC 传承均补充了多组成功、失败与反噬描述。
- 失败后果清晰化：赔偿会显示灵石和暴露变化，拒赔会显示声望、好感、暴露和结仇变化，强行脱身会显示暴露、业力、追踪标记、气血、声望和结仇变化。
- 大比盗术评价细化：盗术痕迹明显时会追加“手脚不净”“旁门留痕”“窃法反噬”“因果缠身”等说明。
- 灵草出售：普通灵草、十年份灵草、三十年份灵草买卖；高年份灵草每月最多正常出售 3 株，超出走黑市。
- 情报值：通过探查情报等行动提升，并影响百药山试炼与斗法台。
- 符箓：护身符、火弹符、避火符、破甲符只作为第一章斗法评分加成，不做复杂装备栏。
- 万魂幡隐藏线：百药山采药有小概率发现隐秘山洞，探索后获得残破魂幡；未获得前不能暗中炼魂。
- 情意锁低阶版：只作为情缘互动时的好感辅助。
- 家族大比 100 分制：测灵问心 25 分、百药山试炼 35 分、斗法台 40 分，并插入 NPC 对手分数表排名。
- 自动模拟测试：可批量模拟不同流派，用于检查第一章数值平衡。
- 数值平衡补丁：降低早期冲榜速度，提高炼丹、坊市符箓和魔道炼魂的代价，并增强模拟评价。
- 结局结算。
- 简单 JSON 存档读档，并兼容旧版存档缺失新增字段的情况。

## 文件说明

- `main.py`：控制台入口、主菜单、行动循环。
- `player.py`：玩家数据、创建、序列化和旧存档字段补全。
- `systems.py`：第一章行动系统、月末事件、坊市商品交易、隐藏线触发。
- `cultivation_assets.py`：灵田、炼丹炉、轻量装备的配置读取、计算和菜单辅助。
- `heishui_market.py`：黑水坊市 v1.1 配置驱动市场、库存刷新、动态价格、服务和风险系统。
- `theft_system.py`：盗术等级、成功率、收益、失败赔偿、月末反噬和大比修正。
- `tournament.py`：家族大比 100 分制评分、NPC 分数表、符箓斗法加成和结局。
- `data.py`：常量、灵根、NPC、坊市商品、文本数据。
- `save_load.py`：JSON 存档读档。
- `configs/cultivation/`：灵田作物、炼丹炉、轻量装备配置。

## 存档

游戏内输入 `9` 或 `S` 可以保存到 `save.json`，输入 `L` 可以从 `save.json` 读取。

## 自动模拟测试

运行：

```bash
python simulate.py
```

默认每个流派模拟 200 局，用于测试各流派数值平衡，不属于正式游玩流程。v0.1.10 起模拟路线包含“盗术投机流”。v0.1.11 起额外输出高阶盗术触发次数、高阶盗术成功次数、月末盗术反噬次数、盗术造成的暴露/业力增量和盗术大比负面标记率。

## 盗术风险提示

盗术是旁门路线。它可以用低阶偷窃补资源，也可能通过功法、修为和机缘形成爆点，但高收益会伴随高暴露、高业力、声望损失、结仇和月末反噬。偷气运、偷因果、偷寿元与偷 NPC 传承均为高门槛、低频率、高风险事件，不适合作为稳定循环收益。
