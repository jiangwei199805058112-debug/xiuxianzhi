# 修仙游戏统一内容母版 v0.1.13+

本文件不是代码，而是后续所有世界观、NPC、剧情、系统设计的总方向。后续新增设定、剧情事件、NPC 档案和系统蓝图，若与本文冲突，应优先以本文为最高优先级 canon，再决定是否修订本文。

当前主 canon：云渊界的核心矛盾是资源与资质的不公，第一章定位为 12 个月内的旁支生存决策与家族大比。

## 一、游戏核心定位

游戏不是传统“一路打怪升级”的爽文修仙，而是：

**资源稀缺下的旁支修仙生存模拟。**

玩家不是天选之子开局，也不是宗门真传开局，而是青岭沈家一个四品灵根旁支。你有一点资质，但不够被家族重视；你有一块劣等灵田、一座破旧炼丹炉、有限的入道资格，以及 12 个月后的家族大比压力。

核心体验是：

> 你每个月只能做有限的事。
> 修炼、种田、炼丹、交朋友、讨好直系、跑坊市、学盗术、接触黑水、冒险采药、压制境界、赌一次厚积薄发。
> 你不可能全拿。每条路都有代价。

这比单纯“正道好、魔道坏”更有意思。世界观里也已经明确，云渊界的核心矛盾不是善恶，而是资源和资质的不公。

## 二、正式世界观口径

### 1. 世界名

正式采用：**云渊界**

“青玄界”暂时不作为正式世界名。可以后续处理为：

| 名称 | 用法 |
| --- | --- |
| 云渊界 | 当前游戏正式世界名 |
| 青玄 | 可作为古称、上古纪元名、某片大陆名，或废案不再使用 |
| 青岭 | 玩家第一章所在区域 |
| 黑水 | 灰色交易体系核心符号 |
| 天罗 | 外部宗门压力核心符号 |

### 2. 世界基调

采用这几个关键词：

**现实主义修仙、低魔起步、人情即资源、算计胜于天赋。**

这意味着游戏前期不要太早出现飞天遁地、神兽仙器、大规模仙魔战争。第一章应该压在“炼气期低阶修士”的真实处境上：

- 灵石少。
- 药材贵。
- 丹炉旧。
- 族学不公平。
- 直系看不起旁支。
- 黑水坊市有机会，但也会坑人。
- 人情关系本身就是资源。
- 很多选择不是“善恶”，而是“值不值得”。

## 三、第一章核心主题

第一章标题建议固定为：

# 第一章：旁支入道

主题词：

**隐忍、积累、选择、代价。**

玩家第一章不是为了立刻成为天下第一，而是解决三个问题：

1. **我能不能活得比普通旁支更好？**
2. **我能不能在家族大比中拿到一个位置？**
3. **我到底要走哪条修仙路？**

第一章结束时，玩家不一定必须拿第一。更重要的是确定身份分支：

| 结局方向 | 含义 |
| --- | --- |
| 家族认可 | 正修、炼丹、厚积薄发等路线较容易触发 |
| 旁支领袖 | 人情、灵田、情报、反压迫路线 |
| 黑水新客 | 坊市、投机、盗术、灰道路线 |
| 被迫出走 | 魔道暴露、盗术败露、得罪族老 |
| 隐忍失败 | 大比落败，但保留第二章散修/黑水/旁支支线 |
| 一鸣惊人 | 厚积薄发或综合路线，震动全族 |

这样即使玩家没有最优操作，也不会只有“失败重开”。这符合真实玩家实际游玩：不会严格按固定流派走，会随意、混合、临时起意地玩。

## 四、玩家开局设定

正式采用：

| 项目 | 设定 |
| --- | --- |
| 身份 | 青岭沈家旁支子弟 |
| 灵根 | 四品灵根，三属性杂灵根 |
| 年龄 | 16-18 岁之间，建议默认 17 岁 |
| 资产 | 一间小屋、一块劣等灵田、一座破旧炼丹炉、少量灵石 |
| 初始功法 | 《青木养气功》或《青木吐纳术》 |
| 初始处境 | 被家族划为“本届大比观察对象”，有入道资格但资源有限 |
| 初始目标 | 12 个月后参加家族大比 |
| 隐藏钩子 | 父母失踪、祖上遗物、旁支旧债、黑水暗线 |

这里和既有设定吻合：玩家有劣等灵田、破旧炼丹炉、模糊入道资格，处于旁支底层但仍有可操作空间。

## 五、第一章 12 个月结构统一版

第一章采用云渊历 1047 年 1-12 月结构。推荐整理成下面这种更适合游戏内容落地的版本：

| 月份 | 主题 | 主要内容 | 关键选择 |
| --- | --- | --- | --- |
| 第1月 | 入道 | 获得灵田、炼丹炉、基础功法，熟悉青岭沈家 | 确定初步方向 |
| 第2月 | 日常 | 灵田种植、认识沈若兰、旁支生活展开 | 修炼还是经营 |
| 第3月 | 黑水初闻 | 第一次听到黑水坊市机会，可选择接触或回避 | 安全正道 / 灰色交易 |
| 第4月 | 小考 | 家族内部小考，沈云庭等直系展示实力 | 低调忍让 / 争取表现 |
| 第5月 | 百药山 | 采药、灵兽、魔修痕迹、沈墨阳线索 | 探险 / 保守 |
| 第6月 | 分化 | 各路线开始明显分化，炼丹、坊市、盗术、关系线展开 | 选择重点培养方向 |
| 第7月 | 冲突 | 沈子岳挑衅、直系压迫、旁支矛盾激化 | 反抗 / 隐忍 / 借刀 |
| 第8月 | 珍稀窗口 | 坊市拍卖、稀有丹药、盲盒、黑水风险 | 投资 / 赌博 / 放弃 |
| 第9月 | 资格审核 | 大比资格可能受影响，声望、业力、关系开始结算 | 保资格 / 走偏门 |
| 第10月 | 人情结算 | NPC 态度逐渐固化，最后的情报、人情窗口 | 站队 |
| 第11月 | 大比前夕 | 闭关、装备、丹药、偷窃、情报、风险事件集中爆发 | 最终准备 |
| 第12月 | 家族大比 | 章末结算，进入不同第二章起点 | 命运分流 |

第 12 月不应该只是简单比武。它应该是整个第一章所有选择的总清算：

- 你有没有丹药？
- 你有没有装备？
- 你有没有人帮？
- 你有没有黑水债？
- 你有没有被直系盯上？
- 你有没有隐藏业力？
- 你有没有压制境界等厚积薄发？
- 你有没有得罪沈若兰、沈霜、沈云庭、沈子岳、沈怀安？
- 你有没有偷过东西但没被发现？

## 六、成长系统统一方向

修炼体系采用这一套：灵根、炼气九层、根基、熟练度、博学度、融会贯通、厚积薄发。

### 1. 境界不是唯一成长

游戏里不能只看“炼气几层”。至少要有这几个成长维度：

| 维度 | 作用 |
| --- | --- |
| 境界 | 基础战力、法力上限、可用功法等级 |
| 根基 | 突破稳定性、同境战力、后期上限 |
| 熟练度 | 炼丹、灵植、盗术、战斗、符箓等技能效率 |
| 博学度 | 解锁隐藏选项、识破陷阱、改良丹方 |
| 人情 | NPC 支援、情报、隐藏剧情 |
| 声望 | 家族、旁支、黑水、天机阁等势力态度 |
| 气运 | 奇遇、捡漏、突破、随机事件权重 |
| 业力 | 心魔、暴露、魔道结局、天道清算 |

### 2. 灵根劣势要能被玩法弥补

玩家四品灵根不是废物，但也不是天才。核心设计应该是：

**直系靠资源和天赋赢前期，玩家靠规划和选择翻盘。**

可翻盘方式包括：

- 厚积薄发。
- 炼丹自给。
- 灵田自产材料。
- 坊市低买高卖。
- 盗术偷取关键资源。
- 情报流提前知道大比对手。
- 人情流获得 NPC 临场帮助。
- 魔道流强行拔高战力但付出代价。

## 七、主要玩法路线统一版

后续路线不要写成死板职业，而是写成“倾向”。玩家可以混合。

| 路线 | 核心玩法 | 强项 | 风险 |
| --- | --- | --- | --- |
| 正修打坐流 | 修炼、打根基、闭关 | 稳、低风险、家族认可 | 前期慢、资源缺 |
| 灵田炼丹流 | 种植、炼丹、自给自足 | 可持续、材料循环强 | 需要时间和熟练度 |
| 坊市符箓流 | 交易、符箓、装备、倒卖 | 灵活、钱多、爆发强 | 被市场波动和黑水牵连 |
| 黑水投机流 | 灰色交易、盲盒、情报 | 高收益、高事件密度 | 暴露、债务、追踪 |
| 盗术流 | 偷资源、偷情报、潜入 | 可越级拿关键物品 | 失败惩罚重、业力高 |
| 魔道炼魂流 | 禁忌功法、速成战力 | 短期极强 | 寿元、业力、NPC 决裂 |
| 厚积薄发流 | 压境、积累、后期爆发 | 大比爆点强 | 前期弱，失败很伤 |
| 人情情报流 | 结交、送礼、解纷、情报 | 分支多、剧情深 | 自身硬实力不足 |
| 随心混合流 | 什么都玩一点 | 真实玩家体验 | 容易资源分散 |

重点是：**随心混合流必须有活路。**

不能让玩家乱玩几个月就必死。它可以拿不到第一，但应该有“前十、被某 NPC 看中、黑水接纳、旁支朋友帮助、第二章散修出走”等出口。

## 八、正道、灰道、魔道边界

这个边界要清楚，因为它决定游戏选择的道德张力。

| 类型 | 代表行为 | 家族态度 | 代价 |
| --- | --- | --- | --- |
| 正道 | 打坐、灵田、炼丹、演武、族学 | 鼓励 | 慢、资源少 |
| 旁门 | 符箓、阵法、丹药速修、坊市投机 | 容忍 | 容易被轻视 |
| 灰道 | 盗术、黑水交易、情报勒索 | 默许但不能公开 | 暴露风险、声望损失 |
| 禁忌 | 炼魂、血祭、吞修为、魔功 | 明令禁止 | 追杀、业力、寿元损耗 |

这里要特别保留一个设计点：

**盗术不是魔道。**

盗术属于灰道。它可以偷灵石、偷丹药、偷情报、偷功法，后期甚至可以偷气运、偷机缘、偷因果，但它不等于炼魂血祭。这样盗术路线才有独立魅力，而不是魔道路线的附属。

## 九、第一章关键 NPC 功能位

先不急着写完整小传，先统一每个 NPC 的“系统职责”。

| NPC | 功能位 | 剧情作用 |
| --- | --- | --- |
| 沈若兰 | 灵田、灵植、人情、温和改革 | 连接药田线、旁支同情线、变异灵草线 |
| 沈云庭 | 直系精英/演武代表/家族未来竞争者 | 展示直系压力，也可成为强敌或认可者 |
| 沈子岳 | 挑衅者、武修对手、可转化竞争者 | 提供第一章明确对抗目标 |
| 沈怀安 | 老辈导师、厚积薄发、家族旧事 | 给玩家稳健路线和禁地线索 |
| 沈霜 | 丹房/盗术/黑水之间的灰色人物，需要二次统一 | 可拆成“冷面丹师”和“盗术少女”两个角色 |
| 沈墨阳 | 魔修残脉、百药山、第二章暗线 | 代表禁忌诱惑和家族旧罪 |
| 天机阁掌柜 | 情报交易 | 提供大比情报、家族丑闻、第二章线索 |
| 黑水商人 | 灰色交易 | 出售盲盒、禁药、残页、赃物回收 |
| 风月楼人物 | 情报、人情、社交 | 不做露骨玩法，定位为情报和关系场 |

沈霜和沈墨阳需要特别处理，因为不同稿子中身份冲突较大。建议是：

- **沈霜保留为冷面丹师**，负责炼丹、丹房、控火、禁丹。
- **盗术少女另起名**，比如“沈鸦”“沈青雀”“沈无痕”，负责盗术教学。
- **沈墨阳保留为被逐出的魔修天才**，不要同时作为族老会大长老。
- **族老会大长老另起名**，比如“沈怀岳”或“沈正衡”。

这样人物功能不会互相打架。

## 十、后续内容开发顺序

建议后面按这个顺序继续写，避免设定发散：

1. **NPC 统一档案**
   先把沈若兰、沈云庭、沈子岳、沈怀安、沈霜、沈墨阳全部重制统一。

2. **第一章主线详细版**
   按 12 个月写每月主事件、可选事件、NPC 事件、路线事件。

3. **盗术系统完整设定**
   包括基础偷窃、高阶偷气运/偷机缘/偷因果/偷寿元/NPC 传承。

4. **炼丹炉与灵田扩展**
   把炼丹炉、灵田、一键种植、作物周期、虫害等内容做成完整系统。

5. **黑水坊市扩展**
   商品、盲盒、天机阁、风月楼、黑水风险、追踪标记、信用系统。

6. **第二章《出走与立身》大纲**
   根据第一章结局分成家族线、宗门线、黑水线、散修线、魔道线。
