## 一、第一章定位

第一章不是传统“新手村打怪升级”，而是：

**旁支子弟在 12 个月内，用有限行动、有限资源、有限人情，决定自己第一条修仙道路。**

玩家开局不是天才真传，而是青岭沈家旁支。玩家拥有：

| 起点资源 | 说明 |
|---|---|
| 四品灵根 | 有修炼可能，但不值得家族重点培养 |
| 一块劣等灵田 | 可作为灵植、炼丹、收入循环的基础 |
| 一座破旧炼丹炉 | 可开启炼丹，但效率和安全性较低 |
| 少量灵石 | 不足以支持全路线发展 |
| 入道资格 | 获得参加本届大比的观察资格 |
| 12个月时间 | 每月行动有限，无法全拿 |

第一章的目标不是“必拿第一”，而是根据玩家选择，决定玩家以什么身份进入第二章。

第一章核心体验：

```text
资源稀缺。
身份被压。
时间有限。
路线很多。
无法全拿。
每个选择都有代价。
```

## 二、事件系统统一规则

本章所有主线、支线、NPC互动和随机事件，统一遵循以下事件规则。

### 2.1 事件 ID 规则

所有事件必须有唯一 ID。

推荐格式：

```text
EVT_CH1_M{月份}_{NPC或系统名}_{序号}_{事件英文短名}
```

示例：

```text
EVT_CH1_M02_SHENZIYUE_001_EXTORT
EVT_CH1_M03_SHENQINGQUE_001_ROOFTOP
EVT_CH1_M05_SHENMOYANG_001_SHADOW
EVT_CH1_M09_SHENZHENGHENG_001_REVIEW
```

规则：

| 规则          | 说明                         |
| ----------- | -------------------------- |
| ID 永不复用     | 废弃事件 ID 不得重新用于其他事件         |
| 与 NPC_ID 联动 | 涉及主要 NPC 的事件 ID 应能看出归属 NPC |
| 可存档         | 事件触发、完成、失败、跳过状态应可被存档记录     |
| 不随中文名改变     | 事件中文名可调整，事件 ID 不变          |

### 2.2 事件状态

| 状态            | 含义          |
| ------------- | ----------- |
| NOT_TRIGGERED | 未触发         |
| ACTIVE        | 进行中         |
| COMPLETED     | 已完成         |
| FAILED        | 已失败         |
| SKIPPED       | 已跳过         |
| LOCKED        | 条件不满足，暂时锁定  |
| DEPRECATED    | 废弃，不再作为开发依据 |

### 2.3 事件触发条件类型

| 类型      | 示例                |
| ------- | ----------------- |
| 时间触发    | 第4月自动触发家族小考       |
| 变量触发    | 盗术痕迹 ≥ 70 触发搜查风险  |
| 前置事件触发  | 触发沈青雀初遇后，才可触发盗术教学 |
| 地点触发    | 夜间进入旁支区触发深夜异动     |
| NPC关系触发 | 沈若兰信任达标触发变异灵草     |
| 路线触发    | 魔道污染达到阈值触发沈墨阳诱惑   |
| 随机触发    | 灵田虫害、黑水行情、旁支求助    |

### 2.4 事件完成/失败要求

每个事件应明确：

| 项目   | 要求               |
| ---- | ---------------- |
| 触发条件 | 玩家何时能遇到该事件       |
| 完成条件 | 玩家怎样算完成          |
| 失败条件 | 玩家怎样算失败          |
| 跳过条件 | 玩家不参与时怎样处理       |
| 变量影响 | 完成/失败/跳过分别改变哪些变量 |
| 后续标记 | 是否开启或关闭后续事件      |

### 2.5 主线事件并发规则

```text
同一时间最多只有一个主线事件处于 ACTIVE。
支线和随机事件可以并存，但不得阻断主线推进。
强制剧情事件不消耗行动点，但必须允许玩家在结果上作出选择。
```

## 三、月度行动点机制

第一章的“时间压力”需要系统化。建议采用月度行动点。

### 3.1 基础规则

| 规则    | 说明                                |
| ----- | --------------------------------- |
| 每月行动点 | 建议基础为 4 点                         |
| 不可累积  | 未使用行动点不能带到下个月                     |
| 强制剧情  | 主线强制剧情通常不消耗行动点                    |
| 自由事件  | 每月可有 1-2 个不消耗行动点的氛围事件             |
| 高关系奖励 | 特定 NPC 高关系可在少数月份提供额外行动机会，但每月最多 +1 |

### 3.2 行动点消耗参考

| 行动类型  |  消耗 | 说明             |
| ----- | --: | -------------- |
| 修炼打坐  | 1-2 | 提升境界、根基或心境     |
| 灵田劳作  |   1 | 维持灵田产出，降低虫害和枯萎 |
| 炼丹    | 1-2 | 消耗药材产出丹药，有失败风险 |
| 百药山采药 |   1 | 获得药材，有受伤和迷路风险  |
| 坊市出行  |   1 | 普通坊市或黑水坊市交易    |
| NPC互动 |   1 | 推进关系、触发个人事件    |
| 盗术行动  |   1 | 高风险获取资源，可能留下痕迹 |
| 特殊剧情  | 0-1 | 根据事件重要性决定      |

设计意图：

```text
玩家不能全拿。
每月必须在修炼、资源、人情、灰道、风险之间取舍。
```

## 五、信息暴露度机制

第一章所有灰色行为，包括盗术、黑水交易、魔道接触、禁丹研究，都会产生“信息暴露度”。

NPC 不是全知。玩家没被发现，就不应立刻触发公开惩罚。

### 5.1 暴露度类型

| 暴露度           | 含义             | 示例          | 后果           |
| ------------- | -------------- | ----------- | ------------ |
| Secret 隐秘     | 无人发现，无痕迹，无追踪标记 | 完美盗取直系灵石    | 无公开后果        |
| Traceable 可追查 | 有人知道出事，但不知道是谁  | 丹房少了一瓶药     | 区域警戒提高       |
| Public 公开     | 多人目击或公开场合发生    | 大比中使用魔功     | 立即触发阵营反应     |
| Leaked 泄露     | 第三方把秘密卖出或传播    | 天机阁出售玩家黑水记录 | 隐秘事件转为可追查或公开 |
| Evidence 证据确凿 | 赃物、人证、痕迹俱在     | 赃物被搜出       | 直接结算惩罚       |

### 5.2 核心规则

```text
未被发现 = 不触发公开声望惩罚、NPC好感惩罚、族老会处罚。
未暴露时，只可能增加痕迹、追踪标记、隐性恶因、物品污染、心理压力等隐藏变量。
公开惩罚只在被发现、被出卖、证据暴露、追踪成功时结算。
沈青雀不是全知 NPC，她只有在亲眼发现、玩家承认、或事件泄露后才会反应。
天机阁也不是全知，除非玩家留下可交易情报、被人目击、或触发追踪标记。
```

### 5.3 痕迹值结算节点

| 结算节点    | 触发条件     | 后果             |
| ------- | -------- | -------------- |
| 第4月小考   | 痕迹值 ≥ 30 | 沈云庭可能多看玩家一眼    |
| 第7月直系挑衅 | 痕迹值 ≥ 50 | 沈子岳挑衅时可能提及玩家异常 |
| 第9月资格审核 | 痕迹值 ≥ 70 | 可能被搜身、盘问或限制行动  |
| 第12月大比  | 痕迹值 ≥ 90 | 对手可能掌握玩家把柄     |

### 5.4 痕迹降低方式

| 方式        | 代价           |
| --------- | ------------ |
| 沈青雀帮忙消除   | 消耗沈青雀好感或信任   |
| 黑水特殊服务    | 消耗灵石和黑水商誉    |
| 暂停灰色行动    | 痕迹随时间自然衰减    |
| 丢弃赃物      | 损失收益，降低证据风险  |
| 使用《敛息藏真诀》 | 消耗灵气、心境或修炼效率 |

## 六、痕迹值衰减与质变

### 6.1 衰减机制

如果玩家连续多月停止灰色行为，痕迹值应逐步下降。

| 情况                | 建议处理                     |
| ----------------- | ------------------------ |
| 当月无盗术/黑水高危/魔道行为   | `theft_trace_level` 小幅下降 |
| 连续 2-3 月老实修炼或经营灵田 | 痕迹明显下降                   |
| 主动消除证据            | 痕迹下降更多                   |
| 赃物仍在身上            | 衰减减慢                     |
| 带追踪标记物品未处理        | 可能不降反升                   |

### 6.2 质变机制

当痕迹值突破高阈值，不应直接 Game Over，而应触发危机事件。

| 痕迹值 | 质变事件                |
| --- | ------------------- |
| ≥50 | 区域警戒提高，部分 NPC 提醒或试探 |
| ≥70 | 第9月资格审核可能搜身         |
| ≥80 | 强制触发高危危机事件          |
| ≥90 | 大比前可能被对手或族老会拿到把柄    |

高危危机事件示例：

| 事件      | 处理方式                |
| ------- | ------------------- |
| 执法堂突击搜查 | 贿赂、丢弃赃物、藏匿、求 NPC 帮忙 |
| 沈子岳堵路质问 | 正面对抗、嘴硬、转移话题、逃跑     |
| 鬼手老陈被查  | 保护他、切割关系、转移证据       |
| 天机阁掌握线索 | 花钱封口、交易情报、承认部分事实    |

设计意图：

```text
灰道不是一碰就死。
但越频繁越危险。
高风险时要让玩家付出代价，而不是直接读档。
```

## 七、章节安全网

第一章必须保证“失败但不死路”。

无论第一章结果如何，以下情况不会发生：

| 不发生事项        | 说明                       |
| ------------ | ------------------------ |
| 主 NPC 在第一章死亡 | 可受伤、疏远、决裂、失踪，但不死亡        |
| 玩家永久废除修为     | 可受伤、中毒、禁闭，但不被永久废修为       |
| 玩家直接游戏结束     | 可出走、逃亡、隐匿，但不直接终局         |
| 核心系统永久关闭     | 核心 NPC 决裂后开启高成本 Fallback |
| 大比缺席导致死档     | 会触发替代事件和第二章入口            |

### 7.1 最差结局保底

最差情况下，玩家仍保留：

| 保底          | 说明                  |
| ----------- | ------------------- |
| 炼气基础修为      | 不被废修为               |
| 最低限度生存能力    | 不直接死亡               |
| 至少一个低度关系    | 通常是沈怀安或普通旁支         |
| 第二章入口       | 可从低位旁支、散修、黑水或逃亡重新开始 |
| Fallback 系统 | 炼丹、灵田、盗术、情报仍有替代渠道   |

## 八、错误补救与惩罚缓冲

### 8.1 错误补救机制

玩家犯错后，应有 1-2 个月补救窗口。

| 错误     | 补救方式                  |
| ------ | --------------------- |
| 盗术暴露   | 帮沈青雀完成高风险任务、处理证据、黑水封口 |
| 魔道轻微污染 | 沈怀安清心法、正道任务、沈若兰灵息调和   |
| 得罪沈霜   | 归还丹方、补偿药材、帮她处理丹房危机    |
| 得罪沈若兰  | 帮助药田、保护变异灵草、停止污染行为    |
| 黑水欠债   | 偿还灵石、完成黑水委托、交出情报      |

### 8.2 惩罚缓冲循环

暴露或失败后不应立即要求玩家读档。

| 失败类型  | 缓冲惩罚                    |
| ----- | ----------------------- |
| 盗窃被抓  | 扣除灵石、没收赃物、禁闭 1-2 月、声望下降 |
| 炼丹炸炉  | 丹炉耐久下降、受轻伤、材料损失         |
| 魔道被怀疑 | 被沈怀安警告、沈正衡观察、心境下降       |
| 黑水违约  | 商誉下降、价格上涨、追债事件          |
| 情缘暴露  | 声誉风险上升，相关 NPC 关系波动      |

极端情况下可触发“绝境逢生”类特殊反馈：

```text
跌入谷底仍能参加大比。
被罚禁闭后带伤上场。
灵石被没收后靠旁支凑药。
被直系羞辱后触发厚积薄发。
```

| 规则          | 说明                        |
| ----------- | ------------------------- |
| 允许偷任何人      | 系统不直接禁止玩家选择               |

| 沈青雀有底线      | 偷救命药、旁支口粮、炼魂物会导致她反感甚至决裂   |
| 盗术不是不劳而获    | 核心是“在不公平世界里拿回一点”，不是无成本刷资源 |

| -------- | ------------------------------------ |
| 魔道前置     | 第5-10月触发过“百药山黑影”或“黑水魔道传闻”            |
| 玩家处于困境   | 资源严重不足、大比胜算低、被族老会压制、被直系羞辱、心境低落至少满足一项 |
| 沈怀安未成功劝阻 | 沈怀安好感不足，或玩家拒绝沈怀安建议                   |
| 玩家仍有欲望   | 玩家没有完全拒绝魔道线                          |

如果玩家资源充足、胜算高、心境稳定，沈墨阳不主动出现。

#### 魔道线代价强化

| 污染阶段    | 后果             |
| ------- | -------------- |
| 魔道污染 ≥3 | 开始出现心魔事件       |
| 魔道污染 ≥5 | 正道 NPC 高度警惕或敌对 |
| 大比中使用魔功 | 若暴露，直接进入逐出/逃亡线 |
| 深度魔染    | 第二章进入魔道/逃亡强绑定  |

### 第12月：家族大比

#### 月份目标

结算第一章所有选择，打开第二章入口。

#### 大比阶段

| 阶段   | 内容       | 受哪些因素影响         |

| 章末分支 | 第二章入口    | 路线总和            |

#### 大比旁观者视角

## 十三、大比缺席替代事件

玩家可能因禁赛、受伤、逃亡或主动弃权缺席大比。缺席不应导致死档。

| 缺席原因    | 替代事件             | 第二章入口       |
| ------- | ---------------- | ----------- |
| 资格被取消   | 旁支区观战，触发“旁支旁观”事件 | 隐忍失败线或旁支改革线 |
| 受伤未愈    | 沈若兰/沈怀安照顾事件      | 药田线、稳健流或散修线 |
| 逃亡      | 第12月变为“逃亡月”      | 被迫出走线       |
| 主动弃权    | 触发 NPC 反应事件      | 根据关系网决定去向   |
| 魔道暴露被禁赛 | 沈墨阳或黑水接触         | 魔道边缘线或逃亡线   |
| 盗术证据确凿  | 族老会处罚、沈青雀支援或切割   | 灰道逃亡线或低位重开  |

## 十四、章末结局分支与变量门槛

> **设计意图**：结局应由玩家 12 个月的综合行为决定。
> **数值参考**：以下门槛为设计参考，后续以模拟验证为准。

### 14.1 家族认可线

参考门槛：

```text
tournament_score ≥ 80
elder_council_suspicion < 50
public_scandal_level < 40
```

结果：

* 获得族学更高权限。
* 沈云庭认可或竞争。
* 沈正衡将玩家列入观察培养。
* 第二章可走家族内部上升线。

第二章阵营反应：

| 阵营  | 反应             |
| --- | -------------- |
| 直系  | 好感/关注上升        |
| 旁支  | 部分羡慕，部分担心玩家被收编 |
| 族老会 | 观察与利用并存        |
| 黑水  | 认为玩家身价提高       |

shen_branch_reputation ≥ 70

## 十五、NPC月度反应机制

每月结束时，系统可根据玩家行为触发轻量 NPC 反应。这类事件通常不消耗行动点。

| 玩家行为    | 可能反应               |
| ------- | ------------------ |
| 本月认真修炼  | 沈怀安表扬或提醒勿急         |
| 本月灵田劳作多 | 沈若兰给予灵植建议          |
| 本月炼丹失败多 | 沈霜冷言点评，但可能教玩家复盘    |
| 本月盗术频繁  | 沈青雀提醒玩家别留下痕迹       |
| 本月黑水交易多 | 鬼手老陈或闻不语试探玩家       |
| 本月心境过低  | 柳听弦或沈怀安可触发安抚事件     |
| 本月魔道接触  | 沈怀安隐约察觉，沈正衡警惕度可能上升 |

设计意图：

```text
NPC 不应只在固定事件中出现。
他们应根据玩家行为产生轻量反馈，让世界更活。
```

## 十七、伏笔回收机制

第一章埋下的主要伏笔，必须在第一章结尾或第二章开头有初步回收。

| 伏笔     | 回收               |
| ------ | ---------------- |
| 沈墨阳黑影  | 大比后或第二章开头正式现身    |
| 沈怀安旧伤  | 第二章成为续命药、禁地或旧事线索 |
| 变异灵草   | 药田改革、黑水争夺或沈若兰线延伸 |
| 鬼手老陈身份 | 黑水线逐步揭露他是沈家旁支弃子  |
| 风月楼隐私  | 第二章情报网或修罗场中回收    |
| 盗术痕迹   | 大比资格审核或第二章追查中回收  |
| 魔道残页   | 沈墨阳或魔修残脉线中回收     |

原则：

```text
伏笔必须有指向，不能不了了之。
不一定第一章全部揭开，但至少要让玩家知道它仍在推进。
```

## 十九、第一章设计边界

### 19.2 第一章可以做

| 可以开放 | 说明             |

### 19.3 成人情缘合规性规则

| 规则      | 说明                    |
| ------- | --------------------- |
