# 二、盗术灰道流 · 12个月事件链

> 路线核心：沈青雀、潜行、偷窃、销赃、痕迹控制、信息暴露度、敛息藏真诀  
> 路线定位：灰道路线，不等于魔道。  
> 设计重点：允许玩家偷，但后果取决于目标、痕迹、暴露、销赃和沈青雀底线。

---

## 2.1 路线总循环

```text
观察目标
→ 判断收益与风险
→ 行动偷取
→ 控制痕迹
→ 隐藏物品
→ 销赃或自用
→ 应对追查
→ 大比前用情报/道具/隐匿形成优势
```

盗术流不是“点一下偷窃就拿资源”，而是一整套风险链：

| 环节   | 失败风险        |
| ---- | ----------- |
| 观察目标 | 目标价值判断错误    |
| 潜入   | 被巡逻发现       |
| 偷取   | 留下痕迹        |
| 藏物   | 赃物被搜出       |
| 销赃   | 被鬼手老陈压价或出卖  |
| 后续追查 | 族老会、黑水、丹房追查 |
| 大比使用 | 被搜身、被对手拿到把柄 |

## 2.4 第1月：旁支失窃传闻

### `EVT_ROUTE_THEFT_M01_001_RUMOR_SEED`

#### 触发条件

第1月旁支区闲谈、坊市闲逛、或沈怀安提醒时可出现。

#### 剧情

旁支区有人说，最近直系主宅外的杂物房少了几块灵石碎片。直系说是旁支偷的，旁支则说直系自己账目不清。

玩家第一次听到“偷”这个字，但还没接触盗术。

#### 选项

| 选择               | 后果        |
| ---------------- | --------- |
| “偷东西终究不好。”       | 正修倾向轻微增加  |
| “直系拿得更多，少几块又怎样。” | 灰道倾向轻微增加  |
| “谁偷的？”           | 开启沈青雀前置印象 |
| 不关心              | 无变化       |

#### 设计意图

第1月不直接开放盗术，只种下“不公平资源结构”的种子。

## 2.5 第2月：被压迫后的选择

### `EVT_ROUTE_THEFT_M02_001_ZIYUE_PRESSURE`

#### 触发条件

沈子岳勒索或羞辱玩家后。

#### 剧情

沈子岳索要灵石，或者在演武场外故意让玩家难堪。玩家可以选择忍、抗、求助，也可以在心里第一次产生“暗中拿回来”的念头。

#### 选项

| 选择        | 后果                        |
| --------- | ------------------------- |
| 忍下        | 心境下降，但安全                  |
| 正面拒绝      | 沈子岳仇恨上升                   |
| 找沈若兰/沈怀安  | 进入正道求助                    |
| 暗中记住他的储物袋 | 开启盗术观察标记                  |
| 打听沈子岳住处   | `theft_access_level` 前置增加 |

#### 设计意图

给盗术路线一个情绪动机：不是“我想犯罪”，而是“我被压迫后想用灰色方式反击”。

## 2.6 第3月：沈青雀登场

### `EVT_ROUTE_THEFT_M03_001_QINGQUE_ROOFTOP`

#### 触发条件

第3月夜间停留旁支区，或玩家已有灰道倾向。

#### 剧情

夜深时，屋顶瓦片轻响。玩家发现沈青雀正在翻找一间空置旁支屋。她没有求饶，而是笑着问：

> “你要喊人吗？”

#### 选项

| 选择      | 后果                     |
| ------- | ---------------------- |
| 放她走     | 沈青雀好感增加                |
| 问她在找什么  | 开启对话                   |
| 帮她望风    | 盗术倾向增加，沈青雀信任增加         |
| 抓她交给族老会 | 沈青雀敌对，盗术线关闭或转 Fallback |
| 跟踪她     | 若成功，发现黑水传闻；若失败，她不信任玩家  |

---

### `EVT_ROUTE_THEFT_M03_002_FIRST_LESSON`

#### 触发条件

玩家没有出卖沈青雀，且沈青雀好感达到初始门槛。

#### 剧情

沈青雀教玩家第一课：

> “偷之前先想三件事：东西值不值，主人惹不惹得起，失败后怎么跑。”

#### 教学内容

| 内容    | 说明                                      |
| ----- | --------------------------------------- |
| 目标价值  | 并非所有目标都值得偷                              |
| 巡逻规律  | 时间、地点、NPC状态影响风险                         |
| 痕迹值   | 没被发现也可能留下痕迹                             |
| 暴露度   | Secret、Traceable、Public、Leaked、Evidence |
| 沈青雀底线 | 不偷救命药、不偷穷旁支最后口粮、不碰魔修炼魂物                 |

#### 解锁

```text
theft_access_level +1
theft_skill +基础
theft_trace_level +少量
```

## 2.7 第4月：第一个小目标

### `EVT_ROUTE_THEFT_M04_001_SMALL_TARGET`

#### 触发条件

盗术基础解锁后。

#### 可选目标

| 目标        | 收益 | 风险    | 沈青雀评价          |
| --------- | -: | ----- | -------------- |
| 直系杂物房灵石碎片 | 低中 | 巡逻低   | “练手刚好。”        |
| 沈子岳随从的小袋  |  中 | 被报复风险 | “胆子不小。”        |
| 旁支老人药钱    |  低 | 道德风险高 | “你敢碰这个，我就不认你。” |
| 丹房边角药材    |  中 | 沈霜怀疑  | “别让我替你背锅。”     |

#### 设计重点

第一次盗术任务必须让玩家明白：

```text
不是收益越高越好。
目标选择本身就是价值观选择。
```

---

### `EVT_ROUTE_THEFT_M04_002_OLD_CHEN_HINT`

#### 触发条件

玩家获得第一批赃物。

#### 剧情

沈青雀提醒玩家，赃物不能随便卖。普通坊市商人不敢收，乱卖会留下证据。她提到黑水坊市有个叫“鬼手老陈”的旧货商。

#### 结果

| 选择   | 后果        |
| ---- | --------- |
| 暂时不卖 | 赃物风险保留    |
| 找老陈  | 开启黑水销赃接口  |
| 自己用  | 可能触发搜身风险  |
| 丢掉   | 损失收益，降低风险 |

## 2.8 第5月：敛息藏真诀

### `EVT_ROUTE_THEFT_M05_001_CONCEALMENT_SCROLL`

#### 触发条件

满足任一：

| 条件        |
| --------- |
| 沈青雀信任达到入门 |
| 鬼手老陈处买到残卷 |
| 黑水盲盒开出    |
| 天机阁情报指向   |

#### 剧情

玩家得到一门灰道辅助法术：

```text
《敛息藏真诀》
```

沈青雀说：

> “真会偷的人，不是手快，是别人连你身上有什么都看不出来。”

#### 解锁效果

| 熟练度 | 效果                   |
| --- | -------------------- |
| 入门  | 隐藏少量小物件气息            |
| 熟练  | 隐藏部分赃物气息，同境界普通探查难发现  |
| 精通  | 压制修为波动，伪装低一层境界       |
| 大成  | 同境界几乎无法探查，可稳定伪装境界    |
| 圆满  | 高一大境界也难以看破，但特殊法宝仍可检定 |

#### 代价

* 消耗灵气。
* 长时间维持降低修炼效率。
* 隐藏魔气增加心境压力。
* 被看破后处罚更重。

## 2.9 第6月：高风险目标

### `EVT_ROUTE_THEFT_M06_001_HIGH_RISK_TARGET`

#### 触发条件

盗术熟练度达到一定值，沈青雀信任不低。

#### 目标池

| 目标      | 收益 | 风险     | 后续        |
| ------- | -: | ------ | --------- |
| 丹房普通药材  |  中 | 沈霜怀疑   | 炼丹材料      |
| 直系账本碎页  | 中高 | 直系追查   | 可获得资源分配情报 |
| 沈子岳训练计划 |  中 | 被报复    | 大比针对      |
| 黑水赃物    |  高 | 追踪标记   | 黑水线深入     |
| 救命药     |  高 | 道德后果极重 | 沈青雀可能决裂   |

#### 事件分支

| 结果       | 后果                        |
| -------- | ------------------------- |
| 完美成功     | 获得资源，无公开惩罚                |
| 成功但留痕    | 获得资源，`theft_trace_level+` |
| 被目击      | 可收买、威胁、逃跑                 |
| 被抓       | 公开丑闻，可能禁闭                 |
| 偷到不该偷的东西 | 开启危机事件                    |

## 2.10 第7月：屋檐下的分赃

### `EVT_ROUTE_THEFT_M07_001_ROOFTOP_SPOILS`

#### 触发条件

共同完成一次灰道任务，沈青雀信任达到中等。

#### 剧情

夜色里，两人躲在旧屋檐下分赃。沈青雀嘴上嫌玩家动作慢，却把最危险的追踪标记留给自己处理。

她问：

> “你以后要是被抓，会先供出我，还是先想办法跑？”

#### 选项

| 选择        | 后果       |
| --------- | -------- |
| “我不会供出你。” | 信任增加     |
| “我会先救你。”  | 情缘/搭档线推进 |
| “看情况。”    | 信任下降     |
| “你也别供出我。” | 灰道伙伴路线推进 |
| 转移话题问追踪标记 | 解锁反追踪教学  |

#### 路线效果

```text
theft_access_level +1
qingque_trust +明显
blackwater_fence_access +可能
```

## 2.11 第8月：带标记的赃物

### `EVT_ROUTE_THEFT_M08_001_MARKED_GOODS`

#### 触发条件

玩家偷黑水物品、买盲盒、或通过老陈销赃。

#### 剧情

玩家发现某件物品灵气波动不自然。沈青雀或老陈提醒：

> “这东西有人盯着，不处理干净，迟早会找上门。”

#### 处理方式

| 方式      | 代价      | 效果    |
| ------- | ------- | ----- |
| 沈青雀处理   | 消耗信任/好感 | 降低追踪  |
| 老陈处理    | 消耗灵石/商誉 | 可能被压价 |
| 敛息藏真诀压制 | 消耗灵气/心境 | 暂时安全  |
| 丢掉      | 损失收益    | 消除风险  |
| 带着不管    | 后续追踪事件  |       |

## 2.12 第9月：资格审核搜查

### `EVT_ROUTE_THEFT_M09_001_QUALIFICATION_SEARCH`

#### 触发条件

`theft_trace_level` 达到中高，或玩家身上有高风险赃物。

#### 审核结果

| 状态        | 后果          |
| --------- | ----------- |
| 痕迹低       | 顺利通过        |
| 痕迹中       | 被盘问，沈正衡警惕增加 |
| 痕迹高但无证据   | 被限制行动，搜身风险  |
| 证据确凿      | 公开处罚，可能取消资格 |
| 有沈青雀/老陈帮助 | 可降低风险       |
| 敛息藏真诀熟练   | 可通过检定       |

#### 重要原则

```text
没有证据，不能直接处罚。
但没有证据不代表安全。
```

## 2.13 第10月：洗手还是深入

### `EVT_ROUTE_THEFT_M10_001_CLEAN_OR_DEEPEN`

#### 触发条件

关系固化阶段，玩家盗术路线已达到中等以上。

#### 分支

| 选择         | 后果              |
| ---------- | --------------- |
| 暂停盗术，消痕    | 痕迹值下降，盗术收益减少    |
| 继续深入       | 盗术熟练提升，高危事件增加   |
| 转向黑水       | 黑水商誉提升，但被控制风险上升 |
| 向沈青雀承诺不碰底线 | 沈青雀信任提升         |
| 为了大比准备最后一票 | 开启第11月高风险偷取     |

#### 设计意图

第10月让玩家明确：

> 盗术不是越偷越无敌，越深入越需要处理后果。

## 2.14 第11月：大比前最后一偷

### `EVT_ROUTE_THEFT_M11_001_STEAL_BEFORE_TOURNAMENT`

#### 触发条件

玩家盗术熟练较高，且没有完全洗手。

#### 目标

| 目标   |      收益 | 风险          |
| ---- | ------: | ----------- |
| 对手情报 | 大比针对性增强 | 被发现后对手警惕    |
| 对手符箓 |    削弱对手 | 证据风险高       |
| 大比名单 |   可避开强敌 | 族老会追查       |
| 丹药材料 |    临时补给 | 沈霜怀疑        |
| 禁忌物品 |     高收益 | 可能触发魔道或黑水危机 |

#### 结果

| 结果     | 后果             |
| ------ | -------------- |
| 完美成功   | 大比获得暗中优势       |
| 成功但留痕  | 第12月可能被对手抓把柄   |
| 失败但逃脱  | 痕迹值大幅增加        |
| 被抓     | 可能禁赛或逃亡        |
| 沈青雀顶风险 | 她可能被牵连，信任或情债加重 |

---

## 2.15 第12月：暗处优势 / 盗术败露

### `EVT_ROUTE_THEFT_M12_001_SHADOW_ADVANTAGE`

#### 触发条件

盗术路线高，且未公开暴露。

#### 大比优势

| 优势      | 说明       |
| ------- | -------- |
| 对手情报    | 知道弱点     |
| 隐藏修为    | 对手误判     |
| 小道具     | 临场反制     |

# 三、炼丹丹房流 · 12个月事件链

> 路线核心：沈霜、炼丹炉、控火、药材、废丹复盘、丹毒、禁丹边界
> 路线定位：资源转化路线，把灵田、采药、坊市材料转化为丹药和大比战力。
> 设计重点：炼丹不是无脑收益，失败、炸炉、丹毒、材料损耗都要存在。

---

## 3.1 路线总循环

```text
获取药材
→ 使用丹炉炼制
→ 失败复盘
→ 提升控火
→ 降低丹毒
→ 获得丹药
→ 大比前爆发
```

炼丹流的核心不是“用药材换丹药”，而是：

| 环节   | 风险        |
| ---- | --------- |
| 获取药材 | 灵石不足、采药受伤 |
| 炼制   | 炸炉、废丹     |
| 服用   | 丹毒、心境波动   |
| 研究丹方 | 禁丹诱惑      |
| 依赖沈霜 | 决裂后效率大幅下降 |
| 大比使用 | 短期强，但可能反噬 |

---

| 变量                          | 说明              |
| --------------------------- | --------------- |
| `alchemy_access_level`      | 炼丹系统开放程度        |
| `alchemy_skill`             | 炼丹熟练度           |
| `fire_control_skill`        | 控火熟练度           |
| `furnace_quality`           | 丹炉品质            |
| `furnace_durability`        | 丹炉耐久            |
| `pill_toxin_level`          | 丹毒              |
| `shenshuang_trust`          | 沈霜信任            |
| `shenshuang_romance_bound`  | 是否进入沈霜情缘绑定      |
| `forbidden_recipe_progress` | 禁丹残页研究进度        |
| `blackwater_recipe_risk`    | 黑水丹方风险          |
| `alchemy_fallback_active`   | 是否启用炼丹 Fallback |

# 四、盗术线与炼丹线的交叉事件

这两条路线很容易交叉，建议专门设计冲突与联动。

---

## 4.1 偷丹房药材

| 条件           | 后果             |
| ------------ | -------------- |
| 盗术成功且未暴露     | 获得药材，丹房库存异常    |
| 留下痕迹         | 沈霜怀疑范围扩大       |
| 被沈霜发现        | 沈霜信任大幅下降       |
| 沈青雀知道偷的是普通药材 | 不赞成但可理解        |
| 偷的是救命药       | 沈青雀强烈反对，沈霜可能决裂 |

---

## 4.2 用盗术取得丹方

| 方式      | 后果             |
| ------- | -------------- |
| 偷普通丹方   | 炼丹提升，但被发现后丹房封禁 |
| 偷禁丹残页   | 禁丹线提前开启        |
| 偷沈霜私人笔记 | 若被发现，沈霜严重敌对    |

# 五、灵田经营流 · 12个月事件链

> 路线核心：沈若兰、灵田、灵植、种子、虫害、变异灵草、药田改革、旁支声望
> 路线定位：稳定材料循环路线，不直接大量产出灵石。
> 设计重点：灵田服务炼丹、药浴、心境、旁支关系和沈若兰情缘线，不能变成无脑刷钱系统。

---

## 5.1 路线总循环

```text
获得劣等灵田
→ 播种/照料
→ 处理虫害、枯萎、灵气不足
→ 收获灵草与材料
→ 供给炼丹/药浴/交易
→ 发现变异灵草
→ 面对直系、黑水、族老会争夺
→ 大比前获得药田支援
```

灵田经营流的核心不是“种田赚钱”，而是建立长期资源循环。

| 环节 | 风险                      |
| -- | ----------------------- |
| 播种 | 种子品质差，成长周期长             |
| 照料 | 占用行动点，收益不立刻兑现           |
| 生长 | 虫害、枯萎、灵气不足              |
| 收获 | 普通灵草居多，高年份灵草少           |
| 变异 | 可能引来直系和黑水关注             |
| 人情 | 帮旁支会提升声望，也会得罪既得利益者      |
| 大比 | 灵田本身不直接战斗，需要转化为丹药、药浴、心境 |

# 六、黑水投机流 · 12个月事件链

> 路线核心：鬼手老陈、闻不语、黑水商誉、盲盒、赃物、行情、债务、追踪标记
> 路线定位：高收益、高不确定、高风险投机路线。
> 设计重点：黑水不是免费商店，而是“有价格、有信用、有债务、有追踪”的灰色生态。

---

## 6.1 路线总循环

```text
听到黑水传闻
→ 接触旧货/情报
→ 小额交易
→ 盲盒或残缺物品
→ 获得高收益或被坑
→ 处理债务、追踪、信用
→ 大比前最后交易
→ 第二章黑水线延伸
```

黑水投机流的核心是：

| 环节   | 风险        |
| ---- | --------- |
| 买货   | 假货、追踪标记   |
| 卖货   | 被压价、留下记录  |
| 借钱   | 债务和利息     |
| 买情报  | 情报不完整或被反卖 |
| 盲盒   | 高波动       |
| 黑水信用 | 守约才有深层资源  |
| 大比使用 | 被搜出或被对手利用 |

# 七、灵田线与黑水线的交叉事件

灵田经营流和黑水投机流会围绕“种子、灵草、变异灵草、黑水买家”产生强交叉。

---

## 7.1 黑水种子污染

| 条件         | 后果            |
| ---------- | ------------- |
| 使用黑水种子且未鉴别 | 作物成长快，但可能污染土壤 |
| 请沈若兰鉴别     | 若关系好，可提前发现问题  |
| 污染未处理      | 第9月药田盘查风险上升   |
| 成功控制污染     | 获得高品质灵草       |

---

## 7.2 变异灵草买家

| 选择       | 后果                 |
| -------- | ------------------ |
| 拒绝黑水买家   | 沈若兰信任上升，黑水兴趣保留     |
| 私下询价     | 黑水商誉可能上升，沈若兰若知道会失望 |
| 出售灵草     | 高灵石收益，药田线背叛        |
| 假意交易钓出买家 | 情报/盗术联动            |
| 交给闻不语估价  | 天机阁掌握灵草秘密          |

---

## 7.3 老陈的旧药田遗种

| 结果    | 说明            |
| ----- | ------------- |
| 真遗种   | 可长出稀有灵草       |
| 假种子   | 浪费行动点和灵石      |
| 带追踪   | 黑水知道玩家种在哪里    |
| 魔染种   | 高收益但污染        |
| 沈若兰鉴别 | 可减少风险，但可能质问来源 |

---

## 7.4 药田盘查与黑水记录

| 状态           | 后果          |
| ------------ | ----------- |
| 灵田异常 + 黑水交易多 | 族老会更容易怀疑    |
| 灵田异常但黑水无记录   | 只算药田线事件     |
| 黑水记录被闻不语保密   | 风险暂缓        |
| 黑水记录泄露       | 药田线变成家族审查事件 |
