## 一、文档定位

本文件用于设计第一章第12月“家族大比”的完整事件结构。

家族大比是第一章高潮，不只是战斗结算，还要承担以下功能：

1. 检验玩家 12 个月路线选择。
2. 展示直系、旁支、黑水、魔道、情报、人情关系的反馈。
3. 让主要 NPC 在看台上对玩家表现做出反应。
4. 让不同路线都能有自己的大比表现方式。
5. 为第二章入口提供清晰分支。
6. 防止玩家因为禁赛、受伤、逃亡等情况进入死路。

## 二、大比设计原则

### 2.1 大比不是单一战斗力测试

大比应同时检查：

| 检查维度 | 代表路线 |
|---|---|
| 修为与根基 | 正修打坐流 |
| 丹药准备 | 炼丹丹房流 |
| 灵草药浴 | 灵田经营流 |
| 符箓装备 | 坊市符箓流 |
| 情报克制 | 情报人情流 |
| 暗中优势 | 盗术灰道流 |
| 黑水资源 | 黑水投机流 |
| 禁忌爆发 | 魔道边缘流 |
| 杂学组合 | 随心混合流 |

### 2.2 胜负不等于唯一结局

大比结果可以分为：

| 结果类型 | 说明 |
|---|---|
| 高排名胜利 | 获得家族重视或外部势力关注 |
| 中等通过 | 保留旁支新星或稳健成长入口 |
| 惨胜 | 排名不错但代价很大 |
| 失利但有表现 | 第二章低起点但不死路 |
| 禁赛/缺席 | 触发替代事件 |
| 暴露逃亡 | 进入黑水/魔道/散修线 |

### 2.3 大比不能只奖励最优解

不同路线都应有发挥空间：

| 路线 | 大比表现方式 |
|---|---|
| 正修 | 稳、抗压、后劲 |
| 灵田 | 药浴、灵草、状态恢复 |
| 炼丹 | 丹药爆发、丹毒风险 |
| 盗术 | 情报、小道具、避开搜查 |
| 黑水 | 旧器、爆发物、债务风险 |
| 情报 | 对手克制、规则理解 |
| 魔道 | 强爆发、高暴露 |
| 坊市 | 符箓、装备、消耗品 |
| 混合 | 多小道具组合、低保 |

## 四、大比前置资格检查

### `EVT_CH1_TOUR_001_ENTRY_CHECK`

| 字段 | 内容 |
|---|---|
| 事件名 | 入场检查 |
| 触发月份 | 第12月 |
| 是否强制 | 是 |
| 作用 | 检查资格、伤势、物品、污染、丑闻 |

### 检查项目

| 检查项 | 变量 | 可能后果 |
|---|---|---|
| 大比资格 | `tournament_qualified` | 未通过则缺席替代 |
| 伤势 | `injury_level` | 高伤势影响发挥 |
| 丹毒 | `pill_toxin_level` | 高丹毒可能反噬 |
| 盗术痕迹 | `theft_trace_level` | 高痕迹触发搜查 |
| 黑水标记 | `blackwater_trace_level` | 标记物可能被扣 |
| 魔道污染 | `demonic_contamination` | 高污染触发沈正衡注意 |
| 公开丑闻 | `public_scandal_level` | 高丑闻影响看台反应 |
| 符箓装备 | `talisman_stock_count` | 数量过多需登记 |

### 处理方式

| 状态 | 结果 |
|---|---|
| 风险低 | 正常入场 |
| 风险中 | 被盘问或限制部分物品 |
| 风险高但无证据 | 入场但沈正衡关注 |
| 证据确凿 | 禁赛、扣押、审查 |
| 魔道明显 | 高危，可能触发逃亡线 |

## 五、大比缺席替代事件

如果玩家无法正常参加大比，不应进入死路。

### 缺席原因与替代事件

| 缺席原因 | 替代事件ID | 替代事件 | 第二章入口 |
|---|---|---|---|
| 资格被取消 | `EVT_CH1_TOUR_ABSENT_001_WATCH_FROM_BRANCH` | 旁支区观战 | 隐忍失败线 / 旁支改革线 |
| 受伤未愈 | `EVT_CH1_TOUR_ABSENT_002_RECOVER_BED` | 病榻听战 | 药田线 / 散修线 |
| 魔道暴露 | `EVT_CH1_TOUR_ABSENT_003_DEMONIC_ESCAPE` | 大比前逃亡 | 魔道逃亡线 |
| 黑水追债 | `EVT_CH1_TOUR_ABSENT_004_BLACKWATER_HIDEOUT` | 藏身黑水 | 黑水债务线 |
| 主动弃权 | `EVT_CH1_TOUR_ABSENT_005_VOLUNTARY_WITHDRAW` | 主动退赛 | 根据关系网决定 |
| 被禁闭 | `EVT_CH1_TOUR_ABSENT_006_LOCKED_ROOM` | 禁闭听钟 | 低起点保底线 |

### 替代事件原则

1. 缺席大比不是游戏结束。
2. 缺席会失去排名奖励。
3. 缺席会触发 NPC 反应。
4. 缺席会影响第二章开局身份。
5. 主NPC不会在第一章因缺席事件死亡。
6. 玩家不会被永久废修为。

## 七、核心对手设计

### 7.1 沈子岳

| 项目 | 内容 |
|---|---|
| 对手ID | `OPP_SHEN_ZIYUE_001` |
| 类型 | 猛攻型 |
| 优势 | 攻势凶、体力强、气势压人 |
| 弱点 | 情绪容易被激怒，防御节奏不稳 |
| 克制方式 | 消耗战、情报激怒、护身符、根基抗压 |
| 关联路线 | 正修、坊市、情报、混合 |
| 失败后反应 | 若堂堂正正击败，可能开始承认玩家 |
| 灰色胜利反应 | 若靠盗术或下黑手获胜，后续仇恨更高 |

#### 战斗事件

`EVT_CH1_TOUR_OPP_001_ZIYUE_CLASH`

| 玩家策略 | 结果倾向 |
|---|---|
| 正面硬抗 | 根基高则有效 |
| 消耗战 | 克制沈子岳 |
| 用符防守 | 可撑过猛攻 |
| 用情报激怒 | 使其露出破绽 |
| 暗中下手 | 可能赢，但有暴露风险 |
| 魔道爆发 | 轻易压制，但高风险 |

### 7.2 沈云庭

| 项目 | 内容 |
|---|---|
| 对手ID | `OPP_SHEN_YUNTING_001` |
| 类型 | 全面型 |
| 优势 | 修为高、资源足、判断稳 |
| 弱点 | 不轻易冒险，可能被规则和局势牵制 |
| 克制方式 | 情报、特殊路线加成、避开硬拼 |
| 关联路线 | 情报、正修、炼丹、灵田 |
| 失败后反应 | 认可玩家潜力，但保持距离 |
| 胜利可能 | 第一章一般不建议正面击败，除非特殊路线和高代价 |

#### 设计原则

沈云庭是第一章上限标尺，不应被普通路线轻松击败。  
玩家可以做到：

1. 让他认真对待。
2. 在某一回合逼出他的底牌。
3. 借规则或情报拿到局部优势。
4. 大幅缩小差距。

沈石头不是反派。他是另一种旁支生存方式：不走捷径，不结交太多人，只靠笨功夫熬。

玩家击败他不应产生恶意，而应产生旁支之间的互相承认。

## 十一、大比动态难度

大比难度不应完全固定，应根据玩家表现动态调整。

### 动态难度触发

| 玩家状态 | 调整 |
|---|---|
| 总分极高 | 增加精英对手或隐藏对手 |
| 路线过强 | 增加对应克制事件 |
| 风险过高 | 增加审查事件 |
| 表现较弱 | 降低初轮难度，保证不死路 |
| 随心混合流 | 提供杂学组合判定 |

### 动态难度原则

1. 动态难度用于增加戏剧性，不用于强行惩罚玩家。
2. 高表现玩家应面对更强挑战，但也应获得更高奖励。
3. 低表现玩家可进入隐忍失败线，不应直接游戏结束。
4. 魔道、盗术、黑水路线的难度主要来自暴露和代价，而非单纯数值压制。

## 十二、大比物品规则

### 物品检查规则

| 物品 | 是否允许 | 备注 |
|---|---|---|
| 普通护身符 | 允许 | 低阶防御符 |
| 低阶攻击符 | 限量允许 | 数量过多影响评价 |
| 清心符 | 允许 | 心境辅助 |

| 盗术藏物 | 禁止 | 搜出后重罚 |
| 伪装符箓 | 禁止 | 罪加一等 |

### 外物依赖评价

| 状态 | 后果 |
|---|---|
| 合理使用 | 正常战术 |
| 频繁使用 | 评价下降，但不违规 |
| 全靠外物 | 排名可能不错，但认可度下降 |
| 使用禁物 | 审查或禁赛 |

## 十九、收束说明

家族大比是第一章的结算器，不是简单战斗关卡。

它应结算：

1. 玩家路线。
2. NPC关系。
3. 风险变量。
4. 暴露度。
5. 大比成绩。
6. 第二章入口。

第一章的设计目标不是让所有玩家都成为第一名，而是让每种选择都有相应结果：

| 玩家类型 | 应有体验 |
|---|---|
| 规划型玩家 | 获得路线高光 |
| 稳健玩家 | 获得低风险认可 |
| 冒险玩家 | 获得高收益与高代价 |
| 乱玩玩家 | 获得混合流保底 |
| 失败玩家 | 进入隐忍失败但不死路 |
| 走偏玩家 | 进入黑水、魔道、逃亡等特殊入口 |
