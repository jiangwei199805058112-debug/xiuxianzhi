# 遭遇事件方向

## 百药山

迷路药径：采药熟练度高时收益更稳。

旧洞府风声：低概率触发魂幡或残页线索。

散修争药：可让步、争夺、交易或偷取。

幼兽踩药：只做氛围或小损失，不扩展灵兽系统。

## 普通坊市

摊主急售：获得低价基础装备或假货风险。

茶棚争执：可听情报，也可能暴露立场。

符箓小户缺货：限制稳定符箓购买。

牙人套话：暴露高时被追问资源来源。

## 黑水坊市

暗巷换牌：首次进入风险提示。

盲盒异响：低概率开出旧物，也可能加追踪。

天机阁换价：情报价格随月份和需求波动。

风月楼听曲：静心或得到小道消息。

## 族学

旧册夹页：功法残页或家族旧案线索。

沈霜提问：根据情报熟练度回应。

拓印风险：盗术流可能偷功法失败。

## 演武场

直系切磋：斗法熟练度和沈云庭关系。

旁支围观：胜负影响声望。

法术失控：受伤或心魔增加。

## 丹房

炉火不稳：炼丹结果文本。

丹香外泄：暴露变化。

沈若兰提醒：丹毒、材料或火候帮助。
