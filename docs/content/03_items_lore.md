# 物品背景

## 古玉瓶

一只温润旧瓶，能催熟灵草。它不像普通法器，更像从某处药园残阵里掉出来的遗物。

风险：催熟痕迹不自然，暴露高时会被族中察觉。

后续：可牵引旧药园、丹师传承或黑水买家。

## 残破魂幡

百药山深处旧洞府遗留之物，能炼化浅层阴气，快速补斗法和修为。

风险：魔气、业力、心魔、暴露。

后续：后山禁地、魔修残脉、沈墨阳线。

## 破旧丹炉

炉壁有裂纹，但比简陋火盆稳定。适合炼丹流早期投资。

## 青铜丹炉

火候更稳，能降低部分丹毒，是丹道入门的重要装备。

## 玄铁丹炉

火力强，成丹多，但动静大。适合有经验的炼丹者，不适合高暴露玩家。

## 轻量装备

武器、法袍、靴子、护符都只提供小幅属性。

装备设计关键词：
- 旧物。
- 基础。
- 克制。
- 可比较。
- 不随机词条。

## 黑水盲盒旧物

来历不明，可能是破损护符、残破法器、假货或追踪媒介。

原则：低概率惊喜，高风险，不稳定循环。
