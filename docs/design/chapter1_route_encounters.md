# 第一章九条路线专属奇遇设计

## 总规则

每条主要路线都有一个“命定奇遇”。奇遇不是随机白送，而是玩家长期行为、路线倾向、NPC关系、风险状态共同触发的路线奖励。

| 项目 | 作用 |
|---|---|
| 触发条件 | 玩家必须真的走这条路，不能随便白拿 |
| 奇遇奖励 | 给路线核心能力或特殊资源 |
| 代价风险 | 防止变成无脑最优 |
| 后续伏笔 | 能接第二章、宗门、黑水、魔道、情缘等内容 |

## 路线摘要

### 正修打坐流：祖祠青灯

触发：多月打坐、心境稳、魔道污染低、沈怀安好感不低。

奖励：清明道心、打坐收益提高、心魔抗性提高、大比稳定性提高。

代价：爆发不足，需要长期投入。

### 灵田经营流：地脉青木窖

触发：多次照料灵田、灵田枯萎率低、沈若兰好感较高。

奖励：《青木长生诀》、小紫灵晶、木系修炼提升、灵田产出稳定。

风险：祖地灵气异常被察觉，紫灵晶来历不明。

### 炼丹丹房流：裂炉生火

触发：多次炼丹，有失败复盘，丹毒未失控，使用旧丹炉。

奖励：炉火感应、炼丹成功率提高、失败保留半成品、丹方推演。

风险：丹毒、火气反噬、炉火需要温养。

### 盗术灰道流：窃运铜钱

触发：跟沈青雀学习盗术，多次藏物、辨痕、偷情报，未公开翻车。

奖励：盗术成功率提高、偷胜机、高阶盗术伏笔。

风险：因果痕迹、沈怀安警觉、灰道势力盯上。

### 黑水投机流：黑契盲盒

触发：多次进入黑水坊市，购买盲盒或黑货，与鬼手老陈有交易。

奖励：半张黑水旧契、特殊交易资格、隐藏商人。

风险：黑水追踪、人情债、第二章黑水债务或追杀。

### 情报人情流：无字竹简

触发：多次买情报，与闻不语建立信用，没有频繁赖账。

奖励：情报成本降低、提前识别风险、大比对手弱点、人情化危机。

风险：人情债、假消息、情报组织视线。

### 魔道边缘流：黑玉残魂

触发：接触魔道残物，有强烈变强执念，沈怀安提醒后仍探查。

奖励：黑玉低语、短期爆发、魔道残篇、极限反杀。

风险：魔道污染、心魔、沈怀安警觉、魔息外泄。

### 坊市符箓流：残阵符匣

触发：多次购买符箓，研究符纸阵盘，坊市消费高。

奖励：符箓失灵率降低、携带上限提高、低阶符箓连击。

风险：烧钱、外物依赖、大比检查限制。

### 随心混合流：杂玉合鸣

触发：接触三系统以上，无单一路线极端专精，无严重丑闻或魔道失控。

奖励：杂学傍身、多系统小幅收益、临场应变、危机补救。

风险：单项爆发不足，高阶选择困难，资源分散。
