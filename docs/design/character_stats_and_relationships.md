# 角色数值与 NPC 关系系统整理

## 一、玩家基础数值

| 数值 | 作用 |
|---|---|
| 姓名 | 主角名字 |
| 月份 | 第一章 12 个月进度 |
| 境界 | 炼气几层、后续筑基等 |
| 修为 | 决定突破、战力、大比表现 |
| 灵石 | 购买丹药、符箓、情报、黑水商品 |
| 灵草/材料 | 炼丹、灵田、任务、交易 |
| 丹药 | 修炼加速、恢复、临时爆发 |
| 符箓 | 大比、战斗、防御、坊市流核心资源 |
| 装备 | 法袍、武器、旧物、盲盒装备等 |
| 当前行动/月度行动 | 记录玩家这个月做了什么 |

---

## 二、修炼相关数值

| 数值 | 作用 |
|---|---|
| 修为值 | 打坐、丹药、奇遇都会增加 |
| 境界层级 | 炼气一层、二层、三层等 |
| 心境 | 越稳越不容易被魔道、流言、失败影响 |
| 心魔/压力 | 太高会触发走火、魔道诱惑、负面事件 |
| 打坐次数 | 判断是否走正修打坐流 |
| 正修路线倾向 | 用于触发“祖祠青灯” |
| 清明道心 | 正修奇遇状态，提升稳定修炼和抗魔能力 |

---

## 三、战斗 / 大比相关数值

| 数值 | 作用 |
|---|---|
| 战力评分 | 影响家族大比排名 |
| 修为评分 | 境界和修为带来的基础分 |
| 符箓评分 | 外物、符箓、阵盘加成 |
| 丹药评分 | 炼丹流、丹药储备加成 |
| 情报评分 | 提前知道对手弱点 |
| 风险扣分 | 魔道、黑水、盗术暴露、丹毒等会扣分 |
| 大比名次 | 第一章结算核心 |
| 前十/前三/第一 | 模拟器统计路线强度用 |

## 四、系统分项数值

### 灵田

| 数值 | 作用 |
|---|---|
| 灵田数量 | 当前灵田规模 |
| 作物状态 | 是否种植、成长阶段 |
| 长势 | 决定收成 |
| 枯萎风险 | 照料不足会提高 |
| 虫害风险 | 触发灵田事件 |
| 灵田照料次数 | 判断是否走灵田经营流 |
| 灵田产出 | 灵草、材料、高年份灵草 |

### 炼丹

| 数值 | 作用 |
|---|---|
| 炼丹次数 | 判断炼丹路线 |
| 炼丹成功率 | 影响成丹 |
| 丹炉品质 | 旧丹炉、好丹炉影响结果 |
| 丹毒 | 吃丹过多或炼丹失败会积累 |
| 废丹数量 | 可用于复盘、事件、材料回收 |
| 炉火感应 | 炼丹奇遇“裂炉生火”的状态 |

### 坊市 / 符箓

| 数值 | 作用 |
|---|---|
| 坊市购买次数 | 判断坊市符箓流 |
| 符箓数量 | 大比和战斗外物 |
| 符箓品质 | 普通、劣质、旧符、黑水符 |
| 符箓失灵风险 | 坊市符箓流风险 |
| 外物依赖 | 过度靠符箓，大比评价受影响 |
| 市场依赖度 | 坊市路线风险变量 |

### 黑水

| 数值 | 作用 |
|---|---|
| 黑水暗号 | 是否能进入黑水坊市 |
| 黑水购买次数 | 判断黑水投机流 |
| 黑水消费 | 模拟器统计黑水投入 |
| 盲盒购买次数 | 黑水盲盒风险 |
| 黑水追踪标记 | 被黑市盯上的风险 |
| 黑水情报日志 | 天机阁、黑水消息 |
| 半张黑水旧契 | 黑水投机流奇遇 |

### 盗术 / 灰道

| 数值 | 作用 |
|---|---|
| 盗术倾向 | 判断是否走盗术灰道流 |
| 盗术痕迹 | 偷窃后留下的隐性风险 |
| 盗术怀疑度 | 被怀疑但未必公开惩罚 |
| 公开丑闻 | 被发现后才增加 |
| 藏物能力 | 隐藏物品、藏修为、避搜查 |
| 偷情报次数 | 高级盗术路线 |
| 窃运铜钱 | 盗术奇遇 |
| 因果痕迹 | 高阶盗术代价 |

原则：偷窃没被发现，不应立刻增加公开惩罚。未暴露时只增加隐性风险、痕迹、因果。

### 魔道

| 数值 | 作用 |
|---|---|
| 魔道污染 | 接触魔功、黑玉残魂后增加 |
| 心魔值 | 压力、执念、魔道诱惑相关 |
| 黑玉残魂持有 | 是否拥有魔道残物 |
| 黑玉使用次数 | 用得越多越危险 |
| 魔息暴露风险 | 大比或事件中可能被发现 |
| 青灯完整度 | 青灯染墨线判断正修是否被污染 |
| 沈怀安警觉 | 正修 NPC 是否发现异常 |

## 五、NPC 关系数值

每个重要 NPC 至少应有：

| 数值 | 说明 |
|---|---|
| 好感度 affection | 喜不喜欢你 |
| 信任度 trust | 愿不愿意把重要事交给你 |
| 敬重 respect | 是否认可你的能力和道心 |
| 警觉 suspicion | 是否怀疑你有问题 |
| 仇恨 hatred | 是否敌对 |
| 人情债 favor_debt | 你欠他，或他欠你 |
| 关系阶段 relation_stage | 陌生、认识、友善、盟友、暧昧、敌对等 |
| 秘密掌握 secret_known | 你是否知道他的秘密 |
| 背叛值 betrayal | 你是否伤害过他 |

示例：

```text
npc_relations = {
  "shen_huaian": {"affection":20,"trust":35,"respect":30,"suspicion":0},
  "shen_qingque": {"affection":10,"trust":15,"gray_path_approval":20,"favor_debt":1}
}
```
