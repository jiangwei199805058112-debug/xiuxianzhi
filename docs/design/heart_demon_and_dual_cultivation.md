# 心魔降低与双修调心系统

## 一、心魔降低手段总表

| 方法 | 降低心魔 | 代价 | 适合路线 |
|---|---:|---|---|
| 静心打坐 | -3~-8 | 消耗行动，慢 | 正修 |
| 清心丹 | -8~-15 | 灵石、丹毒 | 炼丹/坊市 |
| 清心符 | -5~-12 | 消耗符箓，可能失灵 | 符箓流 |
| 祖祠青灯 | 每月 -3 | 需要正修奇遇 | 正修 |
| 柳听弦调心 | -10~-20 | 人情债/灵石/声誉风险 | 情报人情 |
| 沈怀安开解 | -8~-15 | 需要信任 | 正修/回正 |
| 灵田劳作 | -3~-7 | 收益慢 | 灵田 |
| 坦白赎罪 | -10~-30 | 关系/资源/剧情代价 | 灰道/魔道 |
| 休息养神 | -2~-5 | 放弃收益 | 通用 |
| 双修调心 | -8~-35 | 好感、信任、冷却、关系后果 | 情缘/道侣 |

---

## 二、心魔分档

```text
0-30：轻微心乱
31-60：心魔滋生
61-80：心魔缠身
81-100：走火/魔化/重大事件
```

核心原则：心魔低的时候，可以靠静心降；心魔高的时候，必须靠面对。

---

## 三、基础降低方式

### 打坐静心

```text
心魔 -3 ~ -8
心境 +1 ~ +3
本月修炼收益略低，或者消耗一次行动
```

### 清心丹 / 静心丹

```text
心魔 -8 ~ -15
丹毒 +1 ~ +3
灵石消耗
```

### 清心符 / 镇心符

```text
心魔 -5 ~ -12
抵消一次心魔事件
符箓消耗
```

### 祖祠青灯 / 清明道心

```text
每月心魔自然 -3
魔道诱惑抗性提高
走火入魔概率降低
心魔事件判定难度降低
```

## 四、NPC 调心

### 柳听弦调心

```text
心魔 -10 ~ -20
心境 +2
柳听弦好感 +?
可能增加人情债或声誉风险
```

### 沈怀安开解

```text
心魔 -8 ~ -15
魔道污染小幅下降
沈怀安信任 +?
如果隐瞒魔道，则沈怀安警觉 +?
```

### 灵田劳作

```text
心魔 -3 ~ -7
灵田长势 +1
修炼收益降低或消耗行动
```

| 来源 | 降低方式 |
|---|---|
| 偷窃后心魔 | 归还部分赃物、补偿受害者、找沈青雀处理痕迹 |

## 六、双修/亲密调心系统

双修/亲密关系可以作为降低心魔的一种方式，但不能设计成“随便找人做爱 = 心魔清零”。

推荐原则：

> 只有在双方成年、自愿、关系足够、信任足够的情况下，亲密双修才能稳定降低心魔。过程镜头外处理，重点写情绪、信任、灵气交融、关系变化、后果，不写露骨细节。

### 基础效果

普通亲密调心：

```text
心魔 -8 ~ -15
心境 +2
好感 +2
```

信任双修：

```text
心魔 -15 ~ -25
心境 +4
修为 +少量
信任 +2
```

道侣双修：

```text
心魔 -25 ~ -35
心境 +5
修为 +中量
每月心魔自然下降
```

---

## 七、双修限制

```text
好感度 >= 60
信任度 >= 50
双方自愿
双方成年
没有强迫/胁迫/控制
没有严重敌对状态
每月最多 1 次有效双修调心
连续频繁触发收益递减
```

高心魔限制：

| 心魔值 | 双修效果 |
|---|---|
| 0-30 | 很有效，稳定降低 |
| 31-60 | 有效，但需要信任 |
| 61-80 | 只能暂时压制，需要专门事件 |
| 81-100 | 可能反噬双方，必须先镇压心魔 |

---

## 八、双修风险

| 风险 | 说明 |
|---|---|
| 情债 | 对方不是工具人，会产生期待 |
| 占有欲 | 多角色关系会触发修罗场 |
| 声誉风险 | 风月楼、家族、宗门可能有流言 |
| 道侣牵连 | 你走魔道/黑水会影响对方 |
| 心魔传染 | 你心魔太重，可能反伤对方 |
| 双修依赖 | 过度依赖亲密关系压心魔 |
| 背叛惩罚 | 欺骗、利用对方会大幅增加心魔 |
