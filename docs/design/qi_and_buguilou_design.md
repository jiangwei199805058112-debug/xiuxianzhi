# 启与不归楼设定

## 一、基础信息

**角色名：** 启  
**禁名：** 黑魔法之启  
**民间讹称：** 巴啦啦黑魔法·启  
**性别：** 男  
**境界：** 魔界创世神 / 修罗殿至高主宰 / 混沌秩序执掌者  
**所属势力：** 修罗殿  
**表面身份：** 不归楼酒馆老板  
**真实身份：** 魔界之神，三界旧秩序的反叛者  

---

## 二、核心定位

启不是第一章 BOSS。

他是：

| 身份 | 作用 |
|---|---|
| 酒馆老板 | 坊市隐藏 NPC |
| 魔道投资人 | 看中玩家潜力，提前下注 |
| 野心观察者 | 通过对话判断玩家是否有成为魔修/枭雄的可能 |
| 天赋改造者 | 可给玩家洗精伐髓、提升天赋 |
| 心魔诱导者 | 让玩家觉得“正道不公，魔道才真实” |
| 后期救场者 | 第二章危机时暗中救玩家，但让玩家欠下因果 |
| 终局反派伏笔 | 玩家会发现自己早就被他盯上 |

他不急着收玩家，而是先让玩家觉得：这个人虽然危险，但他确实帮了我。

---

## 三、为什么启会帮玩家

| 玩家特质 | 启的看法 |
|---|---|
| 低出身 | 被秩序压迫的人最容易理解魔界 |
| 强野心 | 有欲望，才有可塑性 |
| 不甘心 | 不甘就是魔道最好的种子 |
| 能忍 | 能忍的人，一旦破戒，会走得更远 |
| 多路线接触 | 说明玩家不被单一路线束缚 |
| 心魔潜力 | 心魔不是垃圾，是可利用的力量 |

---

## 四、不归楼

不归楼在坊市深处，却不是每个人都能看见。白天那里只是一堵旧墙，夜里才会出现一扇黑木门。门匾上写着三个字：

**不归楼。**

进了这扇门的人，未必回不来。但回来以后，往往已经不是原来的自己。

---

## 五、首次登场

触发条件可以是：

```text
玩家进入坊市
心魔 >= 20
或者 灵根较低
或者 被沈厉羞辱
或者 大比压力较高
或者 拒绝过黑水诱惑
或者 接触过魔道残物
```

剧情：

玩家在坊市尽头看见一间从未见过的酒馆。门口没有灯。明明坊市很热闹，可那间酒馆周围安静得异常。

酒馆老板红发异瞳，低头擦着怀表。他看见玩家，只说一句：

> “你来得比我看见的未来晚了一点。”

---

## 六、不归楼：第一杯酒

启给玩家倒一杯黑色酒。酒里没有毒，但酒面映出的不是玩家现在的脸，而是未来可能成为的样子：

1. 正修内院弟子
2. 黑水坊市客卿
3. 魔道逃亡者
4. 宗门天骄
5. 三界叛逆者
6. 启身后的魔界门徒

启问：

> “你想成为哪一种？”

| 选项 | 效果 |
|---|---|
| 我只想通过大比 | 启轻笑，给小帮助 |
| 我要变强 | 开启洗精伐髓选项 |
| 我要让他们后悔 | 心魔上升，魔道亲和提高 |
| 我要守住自己 | 启兴趣上升，但不强迫 |
| 我不喝你的酒 | 保持清白，但错过机会 |
| 你到底是谁 | 解锁一点背景伏笔 |

## 七、魔界本源种

第一章不建议直接把玩家改成天灵根。推荐设定为：启不是直接给完整天赋，而是在玩家体内种下一枚“魔界本源种”。

### 伏笔版魔界本源种

```text
修炼效率 +15%
悟性 +1
突破成功率小幅提高
大比评分 +15
心魔不变或小幅 +3
魔道污染不立即增加，或仅隐藏记录
```

隐藏效果：

```text
不归楼印记 = true
启的投资 = 1
未来章节可触发不归楼相关事件
某些高阶存在能感应到异常
```

---

## 八、如果玩家已有祖祠青灯：青灯照夜

若玩家先获得祖祠青灯、清明道心，再接受启的洗精伐髓，可进入特殊状态：

**青灯照夜**

剧情：

启看见沈砚气海里的青灯，停了一瞬。

> “原来已经有人先点了灯。”

沈砚体内青灯没有熄灭，也没有染黑。启只是把一缕本源种在青灯照不到的阴影里。

> “那就看看，是灯照得更远，还是夜藏得更深。”

效果：

```text
保留清明道心
获得不归楼印记
修炼效率提升
暂不增加魔道污染
未来若心魔过高，可能转为青灯染墨
未来若正修坚定，可能转为青灯镇魔
```

## 九、不归楼强行灌顶

**事件名：** 不归楼·强行灌顶  
**触发章节：** 第一章大比前  
**事件性质：** 弱者保底，神秘强化，长期伏笔  

触发条件：

```text
第10月末 / 第11月 / 大比前夜
玩家大比预估排名过低
或 玩家战力评分低于前十门槛
或 玩家修为明显不足
或 玩家连续多月修炼失败
或 玩家心魔、不甘、野心达到一定值
```

剧情摘要：

大比前夜，沈砚听见怀表指针转动。屋中灯火静止，风声停住。他睁眼时已在不归楼。柜台后，红发异瞳的启擦着怀表，只说：

> “你太弱了。”

启说：

> “我原本想让你自己爬上去。可惜，你爬得太慢。”

随后强行点在沈砚眉心，灌入暗紫色灵光，拓宽经脉、洗去杂质、提升修为。

> “这不是恩赐。”
> “这是投资。”

效果：

```text
修为 +大量
境界小概率提升 1 层
修炼效率 +15% ~ +25%
悟性 +1
经脉资质提升
大比评分 +明显提升
身体短期虚弱 1 月
梦中偶尔听见怀表声
不归楼印记 = true
启的投资 = 1
```

## 十、大比后伏笔：怀表余音

大比结束后，不建议立刻强制索债。可以用更含蓄的事件：

玩家夜里梦到不归楼。柜台后空无一人，只有一枚怀表放在桌上。

怀表响了一下。

> 滴答。

然后梦醒。没有任务，没有索债，没有威胁。

状态提示：

```text
你隐约觉得，有人在很远的地方看了你一眼。
```
