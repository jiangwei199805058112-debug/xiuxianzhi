# 青灯染墨：正修破戒分支

## 一、分支定位

如果玩家前期走正修打坐流，并获得祖祠青灯、清明道心，然后第7月接受黑玉残魂诱惑，就进入隐藏分支：

```text
青灯染墨 / 正魔夹心 / 镇魔正修线
```

核心：

> 他明明已经走上正修稳道，明明知道慢慢来也能走下去，但在压力、恐惧、胜负心和资源差距面前，还是伸手接过了魔道捷径。

---

## 二、第7月：黑玉入怀

黑玉残魂低语：

> “你坐了六个月，才走到这里。
> 他们一瓶丹药，一夜便能追上。
> 你所谓的稳，是稳稳地输。”

接受后：

```text
获得：黑玉残简
获得：魔道低语
魔道污染 +15
修炼爆发 +短期提升
清明道心稳定度 -10
沈怀安警觉值 +10
```

因为玩家有祖祠青灯，所以不会直接污染失控，而是出现特殊状态：青灯镇魔。

---

## 三、第8月：灯下黑影

| 力量 | 特征 |
|---|---|
| 祖祠青灯 | 安静、缓慢、稳定 |
| 黑玉残魂 | 锋利、快速、贪婪 |

| 选择 | 结果 |
|---|---|
| 用青灯压制黑玉 | 魔道污染下降，修炼收益正常，黑玉怨念增加 |
| 借黑玉加速修炼 | 修炼收益大幅提高，魔道污染上升 |
| 只研究不使用 | 获得魔道知识，污染小幅上升 |
| 向沈怀安坦白 | 沈怀安好感上升，但黑玉可能被封印 |
| 隐瞒所有人 | 暂时安全，但后续暴露风险上升 |

---

## 四、第9月：沈怀安察觉

沈怀安私下问：

> “你最近修为涨得太快了。”

| 选择 | 结果 |
|---|---|
| 坦白黑玉残简 | 进入青灯镇魔线，沈怀安帮助压制 |
| 半真半假 | 沈怀安怀疑上升，保留黑玉 |
| 完全否认 | 沈怀安警觉大幅上升 |
| 反问沈怀安是否怕自己变强 | 关系下降，魔道污染上升 |
| 请求他别告诉族中 | 沈怀安暂时隐瞒，但要求停止使用 |

沈怀安台词：

> “我不是怕你变强。
> 我是怕你有一天分不清，是你在用它，还是它在用你。”

---

## 五、第10月：青灯染墨

触发条件：

```text
qingming_daoxin = true
demonic_contamination >= 30
black_jade_used >= 2
```

剧情：

沈砚夜里打坐时，气海中的青灯忽然摇晃。灯火没有熄灭，却被一缕黑气缠住。

黑玉残魂低语：

> “你看，所谓正道也不是不能容我。
> 青灯照你，我也照你。
> 只是它让你慢慢走，我让你现在就赢。”

---

## 六、关键分叉

### 青灯镇魔线

```text
魔道污染 -15
清明道心稳定度 +10
短期修炼爆发消失
获得：镇魔心印
沈怀安好感 +15
黑玉残魂怨恨 +1
```

后续：正修为主，魔道为敌。玩家保留黑玉残简，但主要用它理解魔道、克制魔道。

### 借魔修正线

```text
修炼收益 +20%
魔道污染 +10
心魔风险 +5
清明道心稳定度 -5
获得：黑玉吐纳法
沈怀安怀疑 +10
```

玩家心理：

> “我没有杀人。”
> “我没有害同族。”
> “我只是借一点力量。”
> “等大比结束，我就停。”

### 青灯熄灭线

```text
清明道心破损
祖祠青灯状态失效
魔道污染 +35
修炼爆发大幅提高
获得：黑玉魔息
沈怀安关系大幅下降
公开暴露风险上升
```

## 七、大比结局

### 青灯镇魔，正修入前十

```text
保留清明道心
获得镇魔心印
第二章正道评价提高
可开启镇魔路线
```

### 借魔胜敌，青灯半暗

```text
大比名次提高
获得黑玉吐纳法
清明道心变为：半染青灯
沈怀安信任下降
第二章进入正魔摇摆线
```

### 魔息外泄，胜而有污

可能名次保留但被监视，或取消名次进入逃亡/赎罪线。

### 青灯熄灭，转入魔道

```text
正修路线关闭
沈怀安转为追查/阻止
祖祠青灯失效
进入魔道边缘/逃亡/黑水合作线
第二章正道宗门录取受阻
```

推荐主分支：借魔修正，青灯半暗。
