# 成人情缘与双修系统设计

> 版本：v0.1.13+ 草稿
> 用途：为后续情缘系统、双修系统、NPC 好感事件、第一章大比前支援提供事件模板
> 规则：所有可触发亲密关系的角色均为 18 岁以上；所有亲密关系必须基于自愿、好感、信任和剧情条件；亲密过程镜头外处理；重点写情绪、选择、收益、风险和后果。

本文是设计文档，不是运行代码。后续实现时应先参考 `docs/canon/00_content_master.md`，再参考本文。

## 一、设计边界与年龄规则

1. 所有可触发亲密关系的角色必须明确为 18 岁以上。
2. 所有亲密关系必须基于自愿、好感、信任和剧情条件。
3. 亲密过程采用“镜头外处理”，只记录关系标记、收益、风险和后果。
4. 保留成人暧昧、情欲、占有欲、双修收益、声誉风险、修罗场、背叛惩罚、道侣/私奔/公开关系分支等系统设计。
5. 不写露骨性动作、器官、体液、插入、姿势过程。
6. 不加入非自愿、胁迫、精神控制下的亲密内容。
7. 不加入未成年亲密内容。
8. 若 NPC 因剧情处于受伤、昏迷、失控、被控制或无法明确表达意愿的状态，不可触发亲密事件。

## 二、成人情缘统一规则

成人情缘线不是单纯奖励系统，而是长期关系、资源支持、声誉风险和情债因果的组合。

设计原则：

- 情缘推进需要前置事件、好感、信任和路线契合。
- 成人亲密不直接等同于稳定收益，必须伴随声誉风险、占有欲、顾虑或后续责任。
- 双修收益应服务角色主题，例如沈若兰偏灵息调和，沈霜偏丹火控火，沈青雀偏夜行藏息，柳听弦偏心境与情报。
- 亲密后的背叛必须比普通好感下降更严重。
- 多线情缘可以存在，但要有修罗场、流言、信任下降和关系破裂风险。
- 任何情缘 NPC 的帮助都不能让玩家无成本碾压第一章大比。

## 三、关系变量

成人情缘线建议为每个可攻略 NPC 增加以下关系变量。

| 变量 | 说明 |
| --- | --- |
| 好感 | 基础喜欢程度 |
| 信任 | 是否愿意托付秘密、资源和未来 |
| 情欲 | 成人亲密倾向，影响私会和双修触发 |
| 占有欲 | 对玩家与其他 NPC 亲密的敏感程度 |
| 顾虑 | 门第、身份、家规、性格导致的克制 |
| 声誉风险 | 关系被外人发现后的危险 |
| 情债因果 | 亲密后生成的长期关系标记 |
| 道侣倾向 | 是否愿意发展为长期绑定关系 |

## 四、情缘事件等级

| 等级 | 类型 | 说明 |
| --- | --- | --- |
| R1 | 留意事件 | NPC 开始注意玩家 |
| R2 | 亲近事件 | 允许私人对话、赠礼、帮助 |
| R3 | 心动事件 | 出现明显暧昧和独处 |
| R4 | 情缘确认 | 双方确认彼此特殊 |
| R5 | 初次亲密 | 镜头外处理，生成强关系标记 |
| R6 | 双修事件 | 产生修炼收益与风险 |
| R7 | 嫉妒/修罗场 | 多线关系或占有欲触发 |
| R8 | 背叛/决裂 | 亲密后背叛产生严重后果 |
| R9 | 道侣/私奔/公开 | 第二章或后续长期关系分支 |

## 五、沈若兰成人情缘线

沈若兰为 18 岁以上角色。她的情缘线围绕灵田、旁支改革、温和信任和药田秘密展开。她不追求爆发收益，更看重玩家是否尊重灵田、旁支和承诺。

### 事件 R1：巡田留意

**事件ID**：`romance_ruolan_r1_field_notice`
**关联NPC**：沈若兰
**触发时间**：第 2 月起
**触发地点**：灵田区

**触发条件**：

| 条件 | 要求 |
| --- | --- |
| 玩家拥有灵田 | 是 |
| 当月照料灵田 | 是 |
| 未偷取药田灵草 | 是 |
| 沈若兰好感 | >= 10 |

**事件简介**：

沈若兰巡查灵田时，发现玩家的劣等灵田虽然贫瘠，却被照料得很认真。她没有像其他直系那样只看产量，而是问玩家为什么愿意在这种地里花心思。

**玩家选项**：

| 选项 | 结果 |
| --- | --- |
| “地差一些，也总要有人认真种。” | 好感 +8，灵植熟练度 +1 |
| “我只是想多赚点灵石。” | 好感 +3 |
| “旁支没有别的路。” | 好感 +5，开启旁支议题 |
| 敷衍她 | 无变化 |
| 趁机打听药田秘密 | 若博学不足，好感 -3；若博学高，获得药田情报 |

**后续标记**：`ruolan_noticed_player`

## 六、沈霜成人情缘线

沈霜为 18 岁以上角色。她的情缘线围绕丹房、炼丹、控火、信任和冷静反制展开。她不轻易示弱，亲密后对背叛的容忍度很低。

## 十一、修罗场与多情缘后果

多情缘不是禁止项，但必须产生后果。

触发方向：

- 玩家与多个 NPC 达成亲密标记。
- 玩家隐瞒重要关系，被任一情缘 NPC 得知。
- 声誉风险过高，被直系、旁支或黑水传播流言。
- 玩家在关键事件中偏袒一方，导致另一方信任下降。

可能后果：

| 后果 | 说明 |
| --- | --- |
| 好感下降 | 被隐瞒一方降低好感 |
| 信任下降 | 后续托付秘密、资源支援减少 |
| 双修关闭 | 部分 NPC 会暂停或永久关闭双修事件 |
| 声誉风险上升 | 家族和坊市流言影响评价 |
| 情债因果增加 | 后续剧情可能要求补偿、选择或承担责任 |
| 路线冲突 | 灵田、炼丹、盗术、情报线之间出现互斥选择 |

## 十二、情缘背叛惩罚

亲密后的背叛必须比普通好感下降严重。

| 背叛对象 | 惩罚方向 |
| --- | --- |
| 沈若兰 | 药田线断裂，灵田支援消失，可能向族老会沉默不助 |
| 沈霜 | 炼丹支援断裂，可能出现假丹方、丹毒反制 |
| 沈青雀 | 盗术支援断裂，灰道任务不再兜底 |
| 柳听弦 | 风月楼情报涨价，隐私风险上升 |
| 多人同时背叛 | 触发修罗场或声誉崩盘事件 |

背叛类型包括：

- 出卖 NPC 秘密。
- 偷取 NPC 托付之物。
- 利用亲密关系套取资源后转投敌对方。
- 对关键承诺撒谎并被发现。
- 在高风险事件中故意抛弃对方。
