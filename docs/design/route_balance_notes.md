## 一、文档定位

本文件用于约束第一章九条路线的数值强度、风险代价和模拟验收标准。

它不是剧情文档，也不是代码实现文档，而是后续做以下工作时的平衡依据：

1. 接入第一章事件池。
2. 接入家族大比事件。
3. 调整路线收益和风险。
4. 扩展 `simulate.py`。
5. 判断某条路线是否过强或过弱。
6. 判断随心混合流是否仍有生存空间。
7. 判断黑水、盗术、魔道是否变成无成本捷径。

## 二、第一章总体平衡目标

第一章不是让玩家必然夺冠，而是让玩家从旁支底层获得“活下去、被看见、进入第二章”的资格。

### 2.1 总体目标

| 项目 | 目标 |
|---|---|
| 普通玩家 | 能通过若干失败和补救进入第二章 |
| 规划型玩家 | 能打出路线高光 |
| 随心玩家 | 不因路线不纯直接死档 |
| 高风险玩家 | 能获得短期强收益，但必须承担后果 |
| 最优玩家 | 可以冲击高排名，但不应稳定第一 |
| 失败玩家 | 应进入低起点分支，而不是游戏结束 |

### 2.2 第一章结果分层

| 结果层级 | 说明 | 设计目标 |
|---|---|---|
| 第一名 | 极少数情况，强路线+高投入+低失误 | 稀有 |
| 前三名 | 明显成功，获得强关注 | 少量 |
| 前十名 | 第一章主要成功标准 | 中等偏高 |
| 中游 | 有成长但不突出 | 常见 |
| 后段 | 勉强保住资格 | 常见 |
| 缺席/禁赛 | 触发替代入口 | 少量 |
| 逃亡/魔道/黑水债务 | 高风险路线特殊入口 | 低到中 |

## 三、九条路线定位

| 路线 | 核心定位 | 强度来源 | 核心代价 |
|---|---|---|---|
| 正修打坐流 | 稳健正道 | 根基、心境、厚积薄发 | 前期慢，资源少 |
| 灵田经营流 | 材料循环 | 药草、药浴、沈若兰支援 | 行动点占用，直接战力低 |
| 炼丹丹房流 | 资源转化 | 丹药、爆发、沈霜指导 | 丹毒、材料消耗、炸炉 |
| 盗术灰道流 | 灰色收益 | 偷窃、潜行、敛息、销赃 | 痕迹、暴露、道德后果 |
| 黑水投机流 | 高波动投机 | 旧器、盲盒、情报、债务资源 | 假货、追踪、债务 |
| 情报人情流 | 信息布局 | 对手克制、人情、心境 | 信用、人情债、隐私泄露 |
| 魔道边缘流 | 禁忌爆发 | 魔功、禁丹、沈墨阳诱惑 | 污染、心魔、逃亡风险 |
| 坊市符箓流 | 消耗品备战 | 符箓、装备、工具、倒卖 | 灵石压力、外物依赖 |
| 随心混合流 | 真实玩家路线 | 杂学组合、补救、适应性 | 无高阶专精、资源分散 |

## 四、路线强度目标

以下为模拟设计目标，不是固定死数值。实际数值应通过 `simulate.py` 验证后逐步调整。

### 4.1 目标前十率

| 路线 | 目标前十率 | 说明 |
|---|---:|---|
| 正修打坐流 | 60%-75% | 稳但慢，适合作为安全路线 |
| 灵田经营流 | 55%-75% | 需要转化为丹药/药浴才强 |
| 炼丹丹房流 | 45%-70% | 波动较大，成功时强 |
| 盗术灰道流 | 45%-70% | 收益和暴露风险并存 |
| 黑水投机流 | 40%-65% | 高波动，不应稳定 |
| 情报人情流 | 50%-70% | 对战斗直接加成有限，但能避坑 |
| 魔道边缘流 | 35%-65% | 强爆发，高失败/逃亡风险 |
| 坊市符箓流 | 55%-75% | 稳定补短板，但受灵石限制 |
| 随心混合流 | 45%-65% | 有保底，但上限不高 |

### 4.2 目标前三率

| 路线 | 目标前三率 | 说明 |
|---|---:|---|
| 正修打坐流 | 0%-5% | 除非根基极深且低失误 |
| 灵田经营流 | 0%-5% | 单独路线不应强夺前三 |
| 炼丹丹房流 | 3%-10% | 丹药路线可偶尔爆发 |
| 盗术灰道流 | 3%-10% | 暗处优势可冲高，但有风险 |
| 黑水投机流 | 0%-8% | 盲盒/旧器偶尔冲高 |
| 情报人情流 | 0%-6% | 需要配合其他硬实力 |
| 魔道边缘流 | 5%-15% | 爆发强，但代价大 |
| 坊市符箓流 | 1%-8% | 合理备战可冲高，不应稳定 |
| 随心混合流 | 0%-5% | 杂学保底为主，不追求前三 |

### 4.3 目标第一率

| 路线 | 目标第一率 | 说明 |
|---|---:|---|
| 正修打坐流 | 0%-1% | 极少数厚积薄发 |
| 灵田经营流 | 0%-1% | 不应单靠灵田第一 |
| 炼丹丹房流 | 0%-2% | 极少数高丹药爆发 |
| 盗术灰道流 | 0%-2% | 灰色成功但风险极高 |
| 黑水投机流 | 0%-2% | 盲盒/旧器极端成功 |
| 情报人情流 | 0%-1% | 需要硬实力配合 |
| 魔道边缘流 | 0%-3% | 可高爆发，但容易暴露 |
| 坊市符箓流 | 0%-1% | 符箓不应无脑夺冠 |
| 随心混合流 | 0%-1% | 极少数乱中取胜 |

## 五、路线收益上限

### 5.1 正修打坐流

| 收益 | 上限原则 |
|---|---|
| 修为提升 | 稳定但不爆炸 |
| 根基深度 | 可高于其他路线 |
| 心境 | 高稳定 |
| 大比加成 | 第11-12月转化为厚积薄发 |
| 风险 | 低，但前期落后 |

正修流不能变成“什么都不做也最强”。  
它的优势应在后期稳定、大比抗压、低污点审核中体现。

---

### 5.2 灵田经营流

| 收益 | 上限原则 |
|---|---|
| 药材 | 稳定中低产出 |
| 高品质灵草 | 低概率或需长期照料 |
| 药浴 | 大比前重要支援 |
| 灵石 | 不应直接大量产出 |
| 沈若兰支援 | 高关系后明显，但不应单独决定胜负 |

灵田服务材料循环，不是刷钱机器。

---

### 5.3 炼丹丹房流

| 收益 | 上限原则 |
|---|---|
| 丹药 | 能提升修炼和大比表现 |
| 爆气丹 | 高收益，高丹毒 |
| 清心丹 | 心境辅助 |
| 禁丹 | 强但高风险 |
| 沈霜指导 | 高信任后降低失败率 |

炼丹流必须消耗材料、承受失败和丹毒。  
不能让丹药成为无成本稳定加成。

---

### 5.4 盗术灰道流

| 收益 | 上限原则 |
|---|---|
| 灵石/物资 | 中高，但目标决定后果 |
| 情报 | 可辅助大比 |
| 敛息藏真诀 | 强防侦查，但需熟练 |
| 销赃 | 需要黑水或老陈接口 |
| 高阶盗术 | 第一章只能浅尝，不应偷天换命 |

偷窃成功且未被发现时，不应立刻增加公开惩罚。  
但可以增加隐性痕迹、因果、心理压力或后续证据风险。

---

### 5.5 黑水投机流

| 收益 | 上限原则 |
|---|---|
| 旧器 | 可能强，但耐久/追踪风险 |
| 盲盒 | 高波动 |
| 情报 | 准确但昂贵 |
| 债务资源 | 可透支未来 |
| 禁忌物 | 强但高危 |

黑水不能变成“更便宜的普通坊市”。  
黑水必须有假货、债务、追踪、信用和被出卖风险。

---

### 5.6 情报人情流

| 收益 | 上限原则 |
|---|---|
| 对手情报 | 提高针对性 |
| 规则理解 | 降低犯错 |
| 心境调适 | 提高稳定 |
| NPC支援 | 多但不一定深 |
| 人情资源 | 强但有债务 |

情报不应直接替代战斗力。  
它应提高胜率、减少风险、解锁分支，而不是直接给大量分数。

---

### 5.7 魔道边缘流

| 收益 | 上限原则 |
|---|---|
| 战斗爆发 | 第一章最高之一 |
| 修炼速度 | 可短期提高 |
| 禁忌资源 | 强但污染 |
| 沈墨阳诱惑 | 高收益高代价 |
| 逃亡入口 | 特殊结局而非失败 |

魔道不是更容易通关的路线。  
它可以短期强，但必须带来心魔、污染、正道敌意和不可逆风险。

---

### 5.8 坊市符箓流

| 收益 | 上限原则 |
|---|---|
| 护身符 | 提高容错 |
| 攻击符 | 提高爆发 |
| 清心符 | 稳定心境 |
| 旧装备 | 有耐久 |
| 倒卖 | 小额收益 |

符箓流不能靠囤符无脑赢。  
应有灵石压力、库存限制、外物依赖评价和大比检查。

---

### 5.9 随心混合流

| 收益 | 上限原则 |
|---|---|
| 杂学组合 | 提高容错 |
| 误操作补救 | 防止死档 |
| 多NPC低中关系 | 打开多个入口 |
| 适应性天赋 | 第二章潜力 |
| 大比表现 | 保底为主 |

随心混合流不是失败线。  
但它也不能拥有所有路线高阶奖励。

---

## 六、风险变量阈值

### 6.1 通用阈值分层

| 等级 | 数值区间 | 含义 |
|---|---:|---|
| 无 | 0 | 无风险 |
| 轻微 | 1-20 | 只有文本提示或轻微影响 |
| 中等 | 21-50 | 触发警告和小事件 |
| 高 | 51-80 | 触发危机事件 |
| 极高 | 81-100 | 触发强制危机、禁赛、逃亡或重大后果 |

实际代码可使用 0-100，也可使用阶段枚举。  
但文档层面建议统一按 0-100 理解。

---

### 6.2 盗术痕迹 `theft_trace_level`

| 阈值 | 效果 |
|---|---|
| 1-20 | 玩家心虚、偶尔听见风声 |
| 21-50 | 沈青雀提醒，巡逻变多 |
| 51-80 | 出现证据浮出、搜查风险 |
| 81-100 | 盗术危机，可能牵连沈青雀 |

未被发现时，不应直接增加公开声望惩罚。  
只有当证据、目击、出卖或搜查发生后，才转化为 `public_scandal_level`。

---

### 6.3 黑水债务 `blackwater_debt`

| 阈值 | 效果 |
|---|---|
| 1-20 | 老陈口头催账 |
| 21-50 | 黑水要求小任务 |
| 51-80 | 大比前强制交易或还债 |
| 81-100 | 第二章黑水债务人线 |

黑水债务是“透支未来”的机制。  
它可以让玩家第一章暂时变强，但第二章必须还。

---

### 6.4 魔道污染 `demonic_contamination`

| 阈值 | 效果 |
|---|---|
| 1-20 | 黑梦、轻微心魔 |
| 21-50 | 沈怀安察觉，正道NPC不安 |
| 51-80 | 大比中可能外泄，心魔事件增加 |
| 81-100 | 高危，可能逃亡或公开魔道线 |

魔道污染达到高阈值后，不能只当作普通负面状态。  
它应改变 NPC 反应、第二章入口和玩家身份。

---

### 6.5 丹毒 `pill_toxin_level`

| 阈值 | 效果 |
|---|---|
| 1-20 | 轻微口苦，文本提示 |
| 21-50 | 修炼效率下降，沈霜提醒 |
| 51-80 | 大比中可能反噬 |
| 81-100 | 强制调理或严重伤势 |

丹毒应限制炼丹流和坊市买丹流的无限叠加。

---

### 6.6 公开丑闻 `public_scandal_level`

| 阈值 | 效果 |
|---|---|
| 1-20 | 小范围流言 |
| 21-50 | NPC态度变化 |
| 51-80 | 族老会关注，旁支/直系反应 |
| 81-100 | 公开危机，可能影响大比资格 |

公开丑闻和隐性痕迹要分开。  
隐性行为没人发现时，不应凭空变成公开丑闻。

---

### 6.7 外物依赖 `consumable_dependency_level`

| 阈值 | 效果 |
|---|---|
| 1-20 | 合理使用道具 |
| 21-50 | NPC评价“依赖外物” |
| 51-80 | 大比评价下降 |
| 81-100 | 一旦耗尽外物，战力大幅下降 |

用于限制坊市符箓流和黑水旧器流。

---

### 6.8 资源分散 `resource_scatter_level`

| 阈值 | 效果 |
|---|---|
| 1-20 | 正常浅尝 |
| 21-50 | 单路线进展较慢 |
| 51-80 | 触发随心混合流 |
| 81-100 | 大比上限明显下降，但触发保底 |

资源分散不是纯惩罚。  
高资源分散应配合 `mixed_path_score`，触发杂学傍身和适应性天赋。

## 七、大比得分构成

### 7.1 建议得分结构

大比总分 `tournament_score` 可由以下部分组成：

| 项目 | 权重建议 | 说明 |
|---|---:|---|
| 基础修为 | 25% | 境界与灵力 |
| 根基/心境 | 15% | 正修、抗压 |
| 路线专属加成 | 25% | 丹药、符箓、情报、药浴等 |
| 对手克制 | 10% | 情报和策略 |
| NPC支援 | 10% | 高关系奖励 |
| 风险扣分 | -20% 到 0% | 丹毒、暴露、污染等 |
| 临场随机 | ±5% | 小波动 |

### 7.2 路线加成上限

| 路线 | 单路线加成上限 |
|---|---:|
| 正修 | 中高 |
| 灵田 | 中 |
| 炼丹 | 高，但受丹毒限制 |
| 盗术 | 中高，但受暴露限制 |
| 黑水 | 高波动 |
| 情报 | 中，主要克制对手 |
| 魔道 | 很高，但高风险扣分 |
| 坊市 | 中高，但递减 |
| 混合 | 中低，但容错高 |

### 7.3 风险扣分原则

| 风险 | 扣分方式 |
|---|---|
| 丹毒 | 战斗后段扣分 |
| 魔道污染 | 高爆发后高扣分或暴露 |
| 盗术痕迹 | 搜查或证据触发后扣分 |
| 黑水标记 | 物品检查触发后扣分 |
| 外物依赖 | 多轮后扣分 |
| 资源分散 | 上限扣分，但保底提高 |

## 八、路线之间的克制关系

| 路线 | 克制/优势 | 被克制/短板 |
|---|---|---|
| 正修 | 抗压、低风险 | 爆发不足 |
| 灵田 | 持久、恢复 | 直接输出低 |
| 炼丹 | 爆发和修炼 | 丹毒、材料 |
| 盗术 | 情报、小道具、避查 | 公开审查 |
| 黑水 | 奇物爆发 | 假货、追踪 |
| 情报 | 对手克制 | 硬实力不足 |
| 魔道 | 爆发最高 | 正道审查 |
| 坊市 | 稳定补短板 | 灵石和递减 |
| 混合 | 容错、适应 | 无专精上限 |

## 九、黑水、盗术、魔道的收益与代价比例

### 9.1 黑水

| 设计项 | 原则 |
|---|---|
| 收益 | 可以短期超过普通坊市 |
| 成本 | 灵石、债务、信用 |
| 风险 | 假货、标记、追踪、被出卖 |
| 第二章后果 | 债务、人情、黑水绑定 |

黑水收益越高，后续绑定越深。

---

### 9.2 盗术

| 设计项 | 原则 |
|---|---|
| 收益 | 目标不同收益不同 |
| 成本 | 行动点、潜行准备 |
| 隐性风险 | 痕迹、证据、因果 |
| 公开风险 | 被发现后才触发 |
| 道德后果 | 偷弱者/救命物后果严重 |

盗术可以偷任何人，但后果必须不同。  
未发现时不公开惩罚，发现后后果必须足够重。

---

### 9.3 魔道

| 设计项 | 原则 |
|---|---|
| 收益 | 短期战力最高之一 |
| 成本 | 心境、污染、根基 |
| 风险 | 心魔、正道敌意、逃亡 |
| 回头 | 轻度可回，中度高代价，重度近乎不可逆 |
| 第二章后果 | 沈墨阳、魔修残脉、正道审查 |

魔道不能是“更简单通关路线”。  
它是“我愿意付出代价也要赢”的路线。

---

## 十、随心混合流保底标准

随心混合流必须满足以下目标：

| 项目 | 标准 |
|---|---|
| 能进第二章 | 是 |
| 能偶尔前十 | 是 |
| 能稳定前三 | 否 |
| 能触发专属反馈 | 是 |
| 能补救误操作 | 是 |
| 能继承特殊潜力 | 是 |
| 能拿所有路线高阶奖励 | 否 |

### 10.1 Mixed_Path 判定建议

第10月时，如果满足多数条件，可打上 `Mixed_Path`：

| 条件 |
|---|
| 接触过3个以上系统 |
| 没有任一主路线达到高阶专精 |
| 没有深度魔道污染 |
| 没有严重公开丑闻 |
| 没有与核心 NPC 全面决裂 |
| 多数资源投入分散 |

### 10.2 杂学傍身

`jack_of_all_trades_flag` 应提供：

| 效果 | 说明 |
|---|---|
| 初轮保底 | 不容易惨败 |
| 小组合加成 | 低级符+药草+情报组合 |

## 十三、路线过强判定

出现以下情况，应视为路线过强：

| 情况 | 说明 |
|---|---|
| 前十率长期超过90% | 路线过稳 |
| 前三率长期超过25% | 高光过多 |
| 第一率超过5% | 第一章夺冠过易 |
| 风险路线风险事件很少触发 | 代价不足 |
| 黑水/盗术/魔道收益高但惩罚低 | 变成捷径 |
| 坊市符箓靠买符稳定前十 | 外物失控 |
| 灵田直接刷大量灵石 | 经济失控 |
| 情报流无硬实力也稳定高排名 | 情报过强 |
| 随心混合流稳定超过专精流 | 保底过强 |

## 十四、路线过弱判定

出现以下情况，应视为路线过弱：

| 情况 | 说明 |
|---|---|
| 前十率长期低于30% | 路线缺乏可玩性 |
| 大多数模拟死在第一章 | 不符合安全网 |
| 玩家投入路线但没有事件反馈 | 体验弱 |
| 风险路线收益不如低风险路线 | 高风险无意义 |
| 灵田无法支撑炼丹或药浴 | 循环断裂 |
| 炼丹失败率高且无法复盘 | 挫败过强 |
| 盗术未暴露也立刻惩罚 | 逻辑不合理 |
| 魔道轻度接触就死路 | 缺少边缘路线 |
| 随心混合流无法进第二章 | 违背真实玩家目标 |

## 十九、收束说明

第一章的平衡目标不是“所有路线一样强”，而是：

```text
低风险路线更稳。
高风险路线更刺激。
资源路线要有转化成本。
灰色路线要有后果。
魔道路线要有诱惑和不可逆风险。
随心路线要能活，但不能全拿。
```

后续所有第一章玩法实现，都应避免以下两种极端：

1. **最优路线碾压**：玩家只要走某一条固定路线就稳定前十甚至前三。
2. **随心玩家死路**：玩家没有严格规划就无法进入第二章。

理想状态是：

| 玩家类型  | 体验             |
| ----- | -------------- |
| 规划型玩家 | 能打出专属高光        |
| 稳健玩家  | 能低风险进入第二章      |
| 冒险玩家  | 能赌高收益并承担后果     |
| 随心玩家  | 能靠杂学和补救活下去     |
| 失败玩家  | 能进入低起点但可继续的第二章 |
