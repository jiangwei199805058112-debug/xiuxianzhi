# 盗术灰道流分支设计

## 一、路线定位

盗术流不是“到处偷东西赚钱”的低级玩法。

真正的盗术流应是：

> 低阶偷物，中阶偷局，高阶偷命。

主角从“偷回自己的活路”开始，逐步升级到偷情报、偷机会、偷机缘、偷气运、偷因果、偷寿元、偷 NPC 传承。

---

## 二、第一章主题

**主题：从偷活路，到偷命数。**

分歧点：主角发现本该属于旁支的聚气散被主家弟子私吞。第一次不是为了贪婪而偷，而是偷回自己本该得到的东西。

---

## 三、十二个月路线草案

### 第1月：测灵受辱

沈砚测出下品木灵根，被主家少年嘲笑。

### 第2月：第一次失窃

沈砚积攒的灵石或灵草被人偷走，申诉无果。

> “自己看不住东西，也怪不得别人。”

### 第3月：遇见沈青雀

沈青雀先教他“看”：看袖口、脚步、储物袋、药味。

> “盗术第一课，不是偷，是知道谁在偷你。”

### 第4月：偷回自己的月例

| 选择 | 结果 |
|---|---|
| 只拿自己的那份 | 盗术经验少，心境稳定 |
| 拿回所有旁支的份 | 风险上升，旁支暗中感激 |
| 顺手多拿主家的资源 | 收益高，盗术痕迹上升 |
| 留下误导证据嫁祸别人 | 灰道深化，心境压力上升 |

### 第5月：藏物术与伪装境界

获得残术：**《敛息藏形诀》**

初级效果：隐藏储物袋灵气波动、降低搜查概率、同境界不易察觉修为。

高级效果：伪装境界、藏匿特殊物品、黑水标记、魔道气息。

### 第6月：第一次真正偷窃

| 选择 | 结果 |
|---|---|
| 偷走符箓 | 收益直接，风险较高 |
| 只偷符箓清单 | 情报加成，风险较低 |
| 偷换其中一张 | 大比中对手失误，风险中高 |
| 放弃盗术 | 回归正修或混合路线 |

推荐：偷情报，不偷物。

> 低级盗修偷灵石。中级盗修偷法器。高级盗修偷信息。顶级盗修偷命数。

### 第7月：黑水销赃

鬼手老陈提供销赃、伪造来历、洗追踪印、黑市换物、假身份、反查失主背景。

代价：黑水追踪上升、欠老陈人情、被黑市盯上。

### 第8月：偷机缘

主角通过盗术发现灵田阵纹异常，夜里潜入灵田，破开旧阵，得到《青木长生诀》、小紫石、旧木牌。

风险：旧阵失衡，长老察觉祖地灵气异常。

### 第9月：偷气运门槛

获得残缺法器：**窃运铜钱**

弱效果：偷取一点临场气势，让对手第一回合小概率失误，增加因果痕迹。

### 第10月：被发现后的选择

| 选择 | 后果 |
|---|---|
| 赔灵石 | 损失资源，风险下降 |
| 不赔 | 关系下降，持续追查 |
| 反咬诬陷 | 公开冲突 |
| 找沈青雀处理痕迹 | 欠沈青雀人情 |
| 找鬼手老陈买假证据 | 黑水债务/追踪上升 |
| 找闻不语买情报反制 | 情报人情债上升 |

### 第11月：偷大比胜机

| 行为 | 收益 | 风险 |
|---|---|---|
| 偷对手情报 | 大比对该对手加成 | 痕迹小 |
| 偷换符箓 | 对手失误 | 痕迹中 |
| 偷丹药 | 自己续航增加 | 痕迹中高 |
| 改签牌 | 避开强敌 | 暴露后丑闻大涨 |
| 偷气运 | 第一回合优势 | 因果痕迹上升 |

### 第12月：盗术流大比

盗术流的大比不是正面对轰，而是“别人还没上台，就已经输了一半”。

沈砚靠伪装境界、提前知道招式、偷来符箓弱点、藏物术带关键道具、窃运铜钱干扰第一招获胜。

台词：

> “你到现在还以为，我是今天才开始赢你的吗？”

## 四、结局

### 灰道胜利结局

标签：`暗手入局`

第二章进入外门灰市、偷功法、潜入藏经阁、盗取任务情报、黑市悬赏。

### 暴露危机结局

名次保留或削弱，公开声望下降，沈怀安失望，沈青雀带他逃过一劫。

### 高阶盗命伏笔结局

大比后，窃运铜钱上多出一道血线。

沈青雀说：

> “你偷的不是灵石，也不是机会。”

沈砚问：

> “那是什么？”

沈青雀答：

> “是一点命。”
