## 二、核心定位

**一句话概括：**  
例如：他是一个看似温和的族兄，实际上一直在借家族规矩压制旁支弟子的成长。

**反派类型：**

- 权力型反派：追求地位、资源、控制权。
- 生存型反派：为了活下去不择手段。
- 魔道型反派：被魔功、心魔、禁术侵蚀。
- 镜像型反派：和主角走过相似道路，但选择了更极端的方法。
- 情报型反派：不亲自动手，而是操控流言、人情、把柄。
- 灰道型反派：盗术、黑市、赃物、暗线交易高手。
- 悲剧型反派：本可以成为盟友，但被环境推向对立面。

### 结局分支

| 分支 | 条件 | 结果 |
|---|---|---|
| 击败 | 玩家实力/证据足够 | 反派失势或死亡 |
| 放过 | 玩家选择留手 | 反派欠人情或埋下后患 |
| 收服 | 好感/利益/威慑达标 | 成为灰色盟友 |
| 逃走 | 反派保留底牌 | 第二章再登场 |
| 黑化 | 反派彻底失控 | 转入魔道/黑水线 |
| 洗白 | 玩家理解其动机 | 成为复杂盟友 |

## 九、台词模板

**表面客气：** “族弟天资不错，只是这条路，光有天资还不够。”

**暗中威胁：** “有些机会，不是你拿到手，就真属于你的。”

**失败不甘：** “我不是输给了你，我只是输给了他们愿意看见你赢。”

**临死/败退：** “若当年有人给我一次机会，我未必会走到今天。”

**可收服时：** “你若真敢用我，就别指望我变成什么干净人。”
