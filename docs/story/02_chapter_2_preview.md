# 第二章预告

## 注意

第二章当前不实现。本文件只做长期内容蓝图，避免第一章扩展时与未来方向冲突。

## 可能标题

暂定：《外门风起》或《青岭之外》。

## 进入条件

根据第一章结果分流：
- 前十入册：获得家族推荐，进入宗门或家族核心培养。
- 前十蒙尘：成绩足够，但被族老会暗中观察。
- 旁支受查：携带资源异常，需要先处理调查。
- 旁支落榜：通过散修、黑水、人情或补考方式另寻出路。

## 第二章主舞台

候选：
- 宗门外门。
- 沈家核心院。
- 黑水深层坊市。
- 青岭周边散修地带。
- 后山禁地延伸。

## 第二章核心冲突

1. 家族推荐名额有限，直系与旁支矛盾升级。
2. 宗门不完全信任地方家族子弟。
3. 黑水坊市开始追讨旧账或提供更高层交易。
4. 盗术高阶内容产生因果反噬。
5. 魔道残脉试图借玩家复燃。
6. NPC 根据第一章关系进入不同位置。

## 系统延伸方向

- 筑基资源与筑基评价。
- 宗门外门任务。
- 更完整的势力声望。
- NPC 同行、竞争、背叛。
- 功法与技艺配置化。
- 事件池按地点和关系触发。

## 不应过早加入的内容

- 大型宗门管理。
- 自建宗门。
- 复杂装备强化。
- 高阶万魂幡完整系统。
- 复杂寿命系统。
- 大地图开放世界。
