# 盗术系统规划

## 目标

盗术是旁门路线，不是免费资源按钮。它应该带来灵石、材料、情报、功法、修为、气运、机缘、因果、寿元和传承的爆点，也必须带来失败赔偿、声望损失、结仇、业力和月末反噬。

## 分层

低阶：
- 顺手牵羊。
- 夜探摊位。
- 盗取普通情报。

中阶：
- 偷丹药、符箓、装备。
- 偷功法残页。
- 偷黑水情报。

高阶：
- 偷完整功法。
- 偷修为。
- 偷气运。
- 截胡机缘。

禁忌：
- 偷因果。
- 偷寿元。
- 偷 NPC 传承。

## 成功率影响

加成：
- 盗术等级。
- 盗术熟练度。
- 速度、神识、隐匿装备。
- 情报准备。
- 妙手无痕、窥隙探囊等融会状态。

惩罚：
- 目标难度。
- 暴露。
- 业力。
- 结仇。
- 黑水追踪。

## 失败处理

赔偿：损失灵石，暴露小幅增加，后患较轻。

拒赔：声望下降，好感下降，结仇增加。

强行脱身：暴露、业力、追踪标记和气血损失更明显。

## 月末反噬

事件方向：
- 被失主认出。
- 坊市传出盗修传闻。
- 黑水中人盯上。
- 气运反噬。
- 因果缠身。
- 寿元阴债。
- 传承原主追查。

## 大比评价

盗术成功可带来少量隐藏准备或战力收益，但失败、结仇、业力和高阶盗术会带来负面文本。

关键词：
- 手脚不净。
- 旁门留痕。
- 窃法反噬。
- 因果缠身。

## 后续扩展

- 盗术师承。
- 失主追查线。
- 黑水盗修声望。
- 传承原主长期敌对。
- 因果债账本。
