# 炼丹系统规划

## 目标

炼丹系统应补强长期经营路线，让古玉瓶、灵田、炼丹炉和丹药形成循环，但不能成为无风险最强路线。

## 当前组成

- 普通灵草。
- 十年份灵草、三十年份灵草。
- 灵田材料。
- 破旧丹炉、青铜丹炉、玄铁丹炉。
- 丹毒、心魔、暴露。

## 丹炉定位

破旧丹炉：便宜，小幅提高成功率。

青铜丹炉：中等价格，提高成功率并降低丹毒。

玄铁丹炉：较高价格，提高成功率和成丹数量，但动静更大。

## 炼丹过程

炼丹结果应有清晰文本：
- 完美成丹。
- 正常成丹。
- 成丹但有丹毒。
- 部分失败。
- 完全失败。
- 高阶丹炉动静较大。

## 灵田联动

灵田主要提供炼丹材料，不直接产出大量灵石。

灵植熟练度和药理通明可让炼丹材料消耗小概率减少或失败惩罚略降。

## 风险

- 丹毒与心魔。
- 炸炉暴露。
- 古玉瓶暴露。
- 材料不足导致行动收益低。

## 后续扩展

- 丹方系统。
- 丹药品质。
- 丹毒治疗。
- 丹房声望。
- NPC 沈若兰专属炼丹事件。
