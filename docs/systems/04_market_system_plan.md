# 坊市与黑水系统规划

## 普通坊市目标

普通坊市提供稳定但有限的准备资源：
- 丹药。
- 符箓。
- 情报。
- 基础装备。
- 灵草出售。

它不应成为无风险高胜率路线，因此需要价格、库存、收益递减和 NPC 竞争压力。

## 黑水坊市目标

黑水坊市提供灰色机会：
- 天机阁情报。
- 风月楼静心与小道消息。
- 地摊盲盒。
- 低风险黑市线索。
- 高风险来历不明物品。

黑水不是纯惩罚系统，也不是稳定收益系统。它是风险可见的投机入口。

## 经济约束

普通坊市：
- 价格透明。
- 收益中等。
- 风险低。

黑水坊市：
- 价格波动。
- 收益不稳定。
- 追踪风险。

## 事件方向

普通坊市：
- 行情变化。
- 摊主传闻。
- 旧装备。
- 假情报。

黑水坊市：
- 追踪标记。
- 掮客认人。
- 盲盒异物。
- 黑水债主。

## 与路线联动

坊市符箓流重准备。

黑水投机流重风险收益。

随心游玩流低概率接触黑水，不能被迫变成黑水流。

盗术流可偷坊市或黑水相关情报，但风险更高。

## 后续扩展

- 黑水声望。
- 店铺信任等级。
- 拍卖会。
- 假货鉴定。
- 黑水债务。
