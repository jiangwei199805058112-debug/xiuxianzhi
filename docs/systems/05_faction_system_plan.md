# 宗门与势力系统规划

## 当前边界

第一章不实现宗门系统，只保留势力概念。势力影响通过 NPC、事件、大比评价和结局标记表达。

## 第一章势力

- 青岭沈家直系。
- 青岭沈家旁支。
- 药田一脉。
- 演武一脉。
- 族老会。
- 普通坊市。
- 黑水坊市。
- 天机阁。
- 风月楼。
- 散修黑市。
- 魔修残脉。

## 玩家加入或利用方式

家族：通过大比、贡献、好感和清白成长。

药田：灵田、炼丹、沈若兰。

演武：法术训练、沈云庭、斗法台。

黑水：购买、盲盒、情报、追踪。

魔修残脉：魂幡、后山、沈墨阳。

## 第二章宗门方向

宗门系统应从外门开始，不直接做宗门管理。

可做内容：
- 外门任务。
- 宗门贡献。
- 派系好感。
- 试炼排名。
- 筑基资源兑换。

## 风险

势力系统若过早复杂化，会压垮第一章控制台体验。第一步应是事件与声望反馈，不是管理模拟。
